# 血液腫瘤科

---

## 一、正常造血與血球生理

### 1.1 造血系統

- 造血幹細胞（HSC）位於**骨髓**，具自我更新與多系分化能力
- 分化路線：
    - **骨髓系（myeloid）**→ 紅血球、血小板、嗜中性球、單核球、嗜酸性球、嗜鹼性球
    - **淋巴系（lymphoid）**→ B cell、T cell、NK cell
- 成人主要造血部位：扁骨（骨盆、脊椎、胸骨、肋骨、顱骨）
- 胎兒造血順序：卵黃囊（yolk sac, 0–2 月）→ 肝/脾（2–7 月）→ 骨髓（7 月後至終身）
- **髓外造血（extramedullary hematopoiesis）**：骨髓衰竭或纖維化時，肝脾重新造血 → 肝脾腫大

### 1.2 紅血球生理

- 正常壽命 **120 天**，無核、雙凹圓盤
- 生成調控：腎臟分泌 **EPO**（缺氧刺激）→ 促紅血球前驅細胞增殖分化
- 氧氣攜帶：Hb 與 O₂ 結合呈 S 型曲線
    - **右移（P50↑，O₂ 釋放增加）**：↑CO₂、↑H⁺（acidosis）、↑2,3-DPG、↑體溫
    - **左移（P50↓，O₂ 親和力增加）**：↓CO₂、alkalosis、↓2,3-DPG、↓體溫、CO 中毒、fetal Hb（HbF）、methemoglobin
- 正常 Hb 組成：HbA（α₂β₂）97%、HbA₂（α₂δ₂）2–3%、HbF（α₂γ₂）<1%

### 1.3 白血球生理

| 細胞 | 功能 | 臨床意義 |
|------|------|----------|
| 嗜中性球（Neutrophil） | 吞噬、殺菌（先天免疫第一線） | 細菌感染↑；ANC < 500 → 嚴重感染風險 |
| 淋巴球（Lymphocyte） | 適應性免疫（T/B/NK） | 病毒感染↑；HIV → CD4↓ |
| 單核球（Monocyte） | 吞噬、抗原呈遞 | 慢性感染↑ |
| 嗜酸性球（Eosinophil） | 寄生蟲、過敏 | NAACP：Neoplasm, Allergy, Asthma, Collagen vascular disease, Parasites |
| 嗜鹼性球（Basophil） | 過敏反應、IgE 媒介 | CML 時可↑ |

### 1.4 血小板生理

- 由骨髓**巨核細胞（megakaryocyte）**碎裂產生
- 正常壽命 **8–10 天**，正常值 150,000–400,000/μL
- 調控：**TPO（thrombopoietin）**由肝臟合成
- 功能：初級止血 → 黏附（vWF-GPIb）→ 活化（ADP, TXA₂）→ 聚集（fibrinogen-GPIIb/IIIa）

## 二、貧血（Anemia）

### 2.1 定義與分類

- 定義：Hb < 13 g/dL（男）或 < 12 g/dL（女）
- **依 MCV 分類**：

| MCV | 分類 | 常見原因 |
|-----|------|----------|
| < 80 fL | 小球性（Microcytic） | 缺鐵、地中海型貧血、慢性病貧血（部分）、鉛中毒、Sideroblastic anemia |
| 80–100 fL | 正球性（Normocytic） | 急性出血、慢性病貧血、腎衰竭、溶血性貧血、骨髓衰竭 |
| > 100 fL | 大球性（Macrocytic） | B₁₂/葉酸缺乏（megaloblastic）、肝病、酒精、甲狀腺低下、MDS、藥物 |

- **依網狀紅血球（reticulocyte）分類**：
    - Reticulocyte↑（>2%）：出血、溶血 → 骨髓代償增加
    - Reticulocyte↓ 或正常：製造減少（缺鐵、B₁₂/葉酸缺乏、骨髓衰竭、慢性病）

### 2.2 缺鐵性貧血（Iron Deficiency Anemia, IDA）

- **最常見的貧血**
- 原因：慢性失血（消化道出血最重要、月經過多）、攝取不足、吸收不良
- 鐵代謝：十二指腸與近端空腸吸收；儲存為 ferritin/hemosiderin
- 實驗室：

| 指標 | IDA |
|------|-----|
| Serum iron | ↓ |
| TIBC | **↑** |
| Ferritin | **↓**（最敏感且最早變化的指標） |
| Transferrin saturation | ↓（< 20%） |
| MCV | ↓ |
| RDW | **↑** |

- 周邊血抹片：小球性低色素（microcytic hypochromic）、pencil cells、target cells
- 治療：口服鐵劑（ferrous sulfate，空腹吸收最佳、Vit C 促進吸收）；找出並處理出血原因
- **成年男性或停經後女性 IDA → 必須排除消化道惡性腫瘤（大腸鏡/胃鏡）**

### 2.3 慢性病貧血（Anemia of Chronic Disease, ACD）

- 第二常見貧血；慢性感染、自體免疫、惡性腫瘤
- 機制：**Hepcidin↑**（由肝臟因 IL-6 刺激分泌）→ 抑制鐵從腸道吸收及巨噬細胞釋放 → 功能性缺鐵
- 實驗室：

| 指標 | ACD | IDA（比較） |
|------|-----|-------------|
| Serum iron | ↓ | ↓ |
| TIBC | **↓ 或正常** | ↑ |
| Ferritin | **↑ 或正常** | ↓ |
| MCV | 正常或↓ | ↓ |

- 治療：治療原發疾病；嚴重者可用 EPO

### 2.4 巨球性貧血（Megaloblastic Anemia）

- DNA 合成障礙 → 核質不同步（nuclear-cytoplasmic dyssynchrony）

#### 維生素 B₁₂（Cobalamin）缺乏

- 來源：動物性食品
- 吸收：胃壁細胞分泌 **intrinsic factor（IF）** → 與 B₁₂ 結合 → 末段迴腸吸收
- 原因：**惡性貧血（pernicious anemia）**最經典（anti-IF Ab / anti-parietal cell Ab）、胃切除、迴腸疾病/切除、strict vegan、魚肉蟲（Diphyllobothrium latum）
- 表現：
    - 血液：巨球性貧血、**hypersegmented neutrophils**（≥5 葉 ≥ 5%）、pancytopenia
    - **神經症狀**：subacute combined degeneration（後索 + 側索病變）→ 深層感覺↓、vibration/position sense↓、spastic paraparesis、ataxia
    - 精神症狀：dementia、personality change
- 實驗室：B₁₂↓、homocysteine↑、**methylmalonic acid（MMA）↑**（B₁₂ 特異性指標）
- 診斷惡性貧血：anti-IF Ab（高特異性）、Schilling test（已少用）
- 治療：**IM B₁₂ 注射**（口服吸收不良）；治療後需監測低血鉀（造血恢復消耗 K⁺）
- **注意：治療前先確認 B₁₂ 充足再補葉酸，否則葉酸可改善貧血但加重神經病變**

#### 葉酸（Folate）缺乏

- 來源：綠葉蔬菜、水果
- 吸收：近端空腸
- 原因：攝取不足（酒精、營養不良）、需求增加（懷孕、溶血性貧血）、藥物（Methotrexate、Phenytoin、TMP-SMX）、吸收不良
- 表現：與 B₁₂ 類似的血液表現，但**無神經症狀**
- 實驗室：folate↓、homocysteine↑、MMA **正常**（與 B₁₂ 的區別！）
- 治療：口服葉酸補充

### 2.5 地中海型貧血（Thalassemia）

- 球蛋白鏈合成減少 → 不平衡 → 無效造血 + 溶血

#### α-Thalassemia（第 16 號染色體，4 個 α 基因）

| 缺失基因數 | 名稱 | 表現 |
|------------|------|------|
| 1 | Silent carrier（-α/αα） | 正常 |
| 2 | α-thal trait（--/αα 或 -α/-α） | 輕微小球性貧血 |
| 3 | **HbH disease**（--/-α） | 中度貧血，HbH（β₄）inclusion bodies |
| 4 | **Hb Bart's（γ₄）**→ Hydrops fetalis | **致死**，胎兒水腫 |

- HbH 與 Hb Bart's 皆對 O₂ 親和力極高 → 無法有效釋放 O₂

#### β-Thalassemia（第 11 號染色體）

| 類型 | 基因 | Hb 電泳 | 臨床 |
|------|------|---------|------|
| β-thal minor（trait） | β/β⁺ 或 β/β⁰ | **HbA₂↑**（>3.5%） | 輕微小球性貧血，常被誤為 IDA |
| β-thal intermedia | β⁺/β⁺ | HbF↑ | 中度貧血，偶需輸血 |
| β-thal major（Cooley's） | β⁰/β⁰ | **HbF 為主**（60–90%），HbA 極少/無 | 嚴重貧血、出生後 6 個月發病 |

- β-thal major 表現：
    - 嚴重貧血 → 需長期輸血 → **鐵質沉積（iron overload）**→ 心臟衰竭（最主要死因）、肝硬化、內分泌病變、皮膚色素沉著
    - 髓外造血 → 肝脾腫大
    - 骨髓擴張 → 顱骨 X-ray：**hair-on-end appearance（毛刷樣）**、facial bone overgrowth（chipmunk facies）
    - 治療：規律輸血 + **鐵螯合劑（Deferoxamine SC/IV, Deferasirox PO）** + 脾切除（若脾亢進）+ **骨髓移植（唯一根治）**

- **IDA vs β-thal trait 鑑別**：

| 指標 | IDA | β-thal trait |
|------|-----|--------------|
| RDW | **↑** | 正常 |
| Ferritin | ↓ | 正常 |
| HbA₂ | 正常/↓ | **↑** |
| RBC count | ↓ | **↑ 或正常** |
| Mentzer index（MCV/RBC） | > 13 | **< 13** |

### 2.6 Sideroblastic Anemia

- 粒線體內鐵利用障礙 → 鐵堆積在粒線體 → **ring sideroblasts**（Prussian blue 染色可見）
- 原因：先天（ALAS2 基因突變，X-linked）、後天（MDS、酒精、鉛中毒、Isoniazid/INH、chloramphenicol）
- 實驗室：serum iron↑、ferritin↑、transferrin sat↑（鐵過多！）
- 治療：先天型 → **Pyridoxine（Vit B₆）**；後天型 → 治療原因 + 去除致病因子
- INH 引起 → 補 Vit B₆ 預防

### 2.7 再生不良性貧血（Aplastic Anemia）

- 骨髓衰竭 → **全血球低下（pancytopenia）** + 骨髓切片：**脂肪取代（hypocellular/fatty marrow）**
- 原因：
    - 特發性（最常見，多為自體免疫 T cell 攻擊造血幹細胞）
    - 藥物：**Chloramphenicol**（dose-independent，最經典）、benzene、化療
    - 感染：**Parvovirus B19**（pure red cell aplasia，尤其在慢性溶血者引發 aplastic crisis）、hepatitis（seronegative hepatitis → aplasia）、EBV、HIV
    - 先天：**Fanconi anemia**（AR，DNA 修復缺陷，短小、拇指/橈骨異常、café-au-lait spots，↑AML/MDS 風險）
- 治療：
    - 輕度：支持療法、觀察
    - 重度（ANC < 500、plt < 20K、retic < 1%）：
        - < 40 歲 + HLA 配對 → **異體骨髓移植（最佳治療）**
        - 不適合移植 → **ATG + Cyclosporine**（免疫抑制療法）
    - 支持：輸血、感染預防、Eltrombopag（TPO agonist）

### 2.8 溶血性貧血（Hemolytic Anemia）

#### 共通特徵

- 紅血球壽命縮短 → reticulocyte↑、LDH↑、indirect bilirubin↑、haptoglobin↓
- 血管內溶血（intravascular）：hemoglobinemia、hemoglobinuria、hemosiderinuria、haptoglobin 極低
- 血管外溶血（extravascular）：脾臟巨噬細胞吞噬 → 脾腫大
- 周邊血抹片：spherocytes（遺傳性球形紅血球症、AIHA）、schistocytes（MAHA）、target cells（thalassemia、liver disease）、sickle cells

#### 遺傳性球形紅血球症（Hereditary Spherocytosis, HS）

- **AD** 遺傳（最常見），spectrin/ankyrin 等細胞骨架蛋白缺陷
- 表現：貧血、黃疸、脾腫大、**膽色素結石（pigmented gallstones）**
- 診斷：**Osmotic fragility test↑**（球形細胞在低張液中容易破裂）、EMA binding test↓
- **Coombs test（–）**（與 AIHA 區別！）
- 治療：脾切除（最有效，改善但不治癒）；術前需接種疫苗（Pneumococcal, Meningococcal, Hib）
- 合併葉酸補充（慢性溶血消耗增加）

#### G6PD 缺乏症（Glucose-6-Phosphate Dehydrogenase Deficiency）

- **X-linked recessive**（男性發病）
- 機制：G6PD↓ → NADPH↓ → glutathione↓ → RBC 對氧化壓力脆弱
- 觸發：感染、藥物（**Primaquine**、Dapsone、Sulfonamides、Nitrofurantoin）、蠶豆（favism）
- 血抹片：**Heinz bodies**（變性 Hb 沉澱，supravital staining）、**bite cells**（脾臟去除 Heinz body 後）
- 診斷：G6PD 定量（急性溶血期可能偽正常 → 需於恢復期再測）
- 治療：避免觸發因子、支持療法

#### Sickle Cell Disease（鐮刀型紅血球症）

- **HbS**（β⁶ Glu → Val），AR 遺傳
- HbS 在脫氧時聚合 → 紅血球變形鐮刀狀 → 血管阻塞 + 慢性溶血
- 臨床表現：
    - **Vaso-occlusive crisis（疼痛危象）**：最常見，骨痛（dactylitis 在幼兒最早表現）
    - **Acute chest syndrome**：胸痛 + 發燒 + 肺浸潤 → 最常見死因（成人）
    - **Splenic sequestration**：幼兒脾臟急性腫大 + 血紅素驟降 → 可致命
    - **Aplastic crisis**：Parvovirus B19 感染 → reticulocyte 驟降
    - 中風（兒童）、Avascular necrosis（股骨頭）、腎臟（腎乳頭壞死、等張尿）、leg ulcers
    - **Autosplenectomy**：反覆脾梗塞 → 功能性脾缺失 → encapsulated bacteria 感染↑（Strep pneumoniae, H. influenzae）→ **Howell-Jolly bodies** 出現
- 診斷：Hb 電泳（HbS 為主）、Sickle solubility test、周邊血抹片（sickle cells、target cells、Howell-Jolly bodies）
- 治療：
    - 急性疼痛：hydration、止痛（NSAIDs → opioids）、O₂
    - **Hydroxyurea**：增加 **HbF** → 減少鐮形化，減少疼痛危象
    - 預防：Penicillin prophylaxis（至少到 5 歲）、疫苗、葉酸
    - 根治：**骨髓移植**
    - **Sickle cell trait（HbAS）**：通常無症狀，但高海拔/極度脫水可誘發；腎髓質梗塞 → 等張尿/血尿

#### 自體免疫溶血性貧血（Autoimmune Hemolytic Anemia, AIHA）

| 類型 | 抗體 | Coombs test | 溫度 | 相關疾病 |
|------|------|-------------|------|----------|
| **Warm AIHA** | IgG | DAT (+), IgG | 37°C | **SLE**、CLL、Methyldopa、Penicillin |
| **Cold agglutinin** | IgM | DAT (+), C3 | < 30°C | Mycoplasma、EBV（infectious mono）、Lymphoma |

- Warm AIHA 最常見（70–80%）
- 治療：Warm → 類固醇（首選）→ Rituximab → 脾切除；Cold → 保暖、避免冷暴露、Rituximab（脾切除效果差）
- **Direct Coombs test（DAT）**：檢測紅血球表面有無抗體/補體
- **Indirect Coombs test（IAT）**：檢測血清中有無游離抗體（用於交叉配對、懷孕篩檢）

#### 微血管病性溶血性貧血（Microangiopathic Hemolytic Anemia, MAHA）

- 紅血球通過受損微血管 → 機械性破壞 → **schistocytes（碎片紅血球）**
- 見於：TTP、HUS、DIC、HELLP syndrome、惡性高血壓、人工瓣膜

#### Paroxysmal Nocturnal Hemoglobinuria（PNH）

- **後天性造血幹細胞突變**（PIGA 基因）→ GPI anchor 缺失 → CD55（DAF）、CD59（MIRL）缺乏 → 補體媒介溶血
- 臨床三徵：**血管內溶血**（晨起深色尿）、**血栓**（最常見死因，尤其 hepatic/Budd-Chiari）、**骨髓衰竭**（可與 aplastic anemia 互相轉換）
- 診斷：**Flow cytometry**（CD55/CD59 缺失）→ gold standard（取代 Ham's test / sucrose lysis test）
- 治療：**Eculizumab**（anti-C5 monoclonal Ab）→ 抑制末端補體 → 需接種 meningococcal vaccine（C5 阻斷 → N. meningitidis 感染風險）
- 唯一根治：骨髓移植

## 三、白血球疾病

### 3.1 急性白血病（Acute Leukemia）

- 共通特徵：骨髓中 **blast ≥ 20%**（WHO 定義）
- 臨床表現：骨髓衰竭 → 貧血（疲倦）、嗜中性球低下（感染）、血小板低下（出血）

#### 急性骨髓性白血病（Acute Myeloid Leukemia, AML）

- 好發：**成人**（中位年齡 65 歲）
- 血抹片/骨髓：**Auer rods**（特徵性，嗜天青顆粒結晶）
- 細胞化學染色：**MPO（myeloperoxidase）(+)**、Sudan Black B (+)
- Flow cytometry：CD13, CD33, CD34, CD117
- 重要亞型：

| FAB 分型 | 名稱 | 特點 |
|----------|------|------|
| M0–M1 | Undifferentiated / Minimally differentiated | — |
| M2 | AML with maturation | **t(8;21)**，最常見；預後較好 |
| **M3** | **APL（Acute Promyelocytic Leukemia）** | **t(15;17) PML-RARα**；Auer rods 密集（faggot cells）；**DIC**為主要死因 |
| M4 | Acute myelomonocytic | inv(16)（Eo 亞型，預後好） |
| M5 | Acute monocytic | 皮膚/牙齦浸潤；高白血球→ leukostasis |
| M6 | Acute erythroid | — |
| M7 | Acute megakaryoblastic | **Down syndrome 相關**；骨髓纖維化 |

- **APL（M3）為最重要**：
    - t(15;17) → PML-RARα 融合基因
    - 極易併發 **DIC**（急性期致死主因）
    - 治療：**ATRA（all-trans retinoic acid）+ ATO（arsenic trioxide）** → 誘導分化 → 治癒率最高（>90%）
    - **Differentiation syndrome（formerly ATRA syndrome）**：ATRA 使用後 → 發燒、呼吸困難、肺浸潤、pleural/pericardial effusion、weight gain → 治療：**Dexamethasone**
    - 不可單用 ATRA（會 relapse）

- AML 治療原則：
    - Induction：**7+3（Cytarabine 7 天 + Daunorubicin 3 天）**
    - Consolidation：high-dose Cytarabine 或 allo-HSCT
    - 預後好：t(8;21)、inv(16)、t(15;17)、NPM1 mutation
    - 預後差：complex karyotype、FLT3-ITD、TP53 mutation、MDS-related

#### 急性淋巴芽球白血病（Acute Lymphoblastic Leukemia, ALL）

- 好發：**兒童**（2–5 歲 peak，最常見兒童癌症）
- 細胞化學：MPO (–)、**PAS (+)**、**TdT (+)**（terminal deoxynucleotidyl transferase）
- Flow cytometry：
    - B-ALL（最常見，75%）：CD10（CALLA）、CD19、CD20、CD22
    - T-ALL：CD2, CD3, CD5, CD7 → 常以前縱膈腔腫塊表現（thymic mass）

- 重要細胞遺傳學：

| 染色體異常 | 意義 |
|------------|------|
| **Hyperdiploidy（>50 chromosomes）** | **預後最好**（兒童） |
| t(12;21) TEL-AML1 | 預後好（兒童最常見） |
| **t(9;22) BCR-ABL（Philadelphia chromosome）** | **預後最差**（成人 ALL 30%，兒童 3%），需加 TKI |
| t(4;11) MLL | 嬰兒 ALL，預後差 |
| Hypodiploidy（< 44） | 預後差 |

- 治療：多藥化療（Induction → Consolidation → Maintenance）
    - Induction：Vincristine + Prednisone + Daunorubicin + Asparaginase ± Cyclophosphamide
    - Maintenance：**6-MP（6-Mercaptopurine）+ Methotrexate**（ALL 特有，需維持 2–3 年）
    - **CNS 預防**：intrathecal Methotrexate ± cranial radiation（ALL 易侵犯 CNS）
    - Ph(+) ALL → 加 **Imatinib** 或其他 TKI
- 兒童 ALL 治癒率 **> 85–90%**；成人較差（40–50%）

### 3.2 慢性白血病

#### 慢性骨髓性白血病（Chronic Myeloid Leukemia, CML）

- **t(9;22) Philadelphia chromosome → BCR-ABL 融合基因**（constitutive tyrosine kinase）
- 好發 40–60 歲
- 三期病程：
    1. **Chronic phase**（85% 在此期診斷）：白血球計數極高（可 >100K）、嗜鹼性球↑、**各階段骨髓細胞皆有（whole spectrum）**、巨脾
    2. **Accelerated phase**：blast 10–19%、嗜鹼性球↑≥20%、症狀加重
    3. **Blast crisis**：blast ≥ 20% → 行為如急性白血病（2/3 轉為 AML，1/3 轉為 ALL）
- 血液：WBC 極高、左移（left shift，各階段皆見）、**嗜鹼性球增多（basophilia）**
- **LAP score 低**（vs. leukemoid reaction → LAP 高）
- 骨髓：hypercellular
- 診斷：BCR-ABL by FISH/RT-PCR + 周邊血抹片
- 治療：
    - **Imatinib（Gleevec）**：第一代 TKI → 標靶 BCR-ABL → 革命性藥物
    - 二代：Dasatinib、Nilotinib；三代：Ponatinib（用於 T315I 突變抗藥）
    - **Allo-HSCT**：唯一根治但風險高，用於 TKI 失敗或 blast crisis
- 預後：TKI 時代 10 年存活率 > 80%

#### 慢性淋巴球白血病（Chronic Lymphocytic Leukemia, CLL）

- **最常見的成人白血病**（西方國家，台灣相對少見）
- B cell neoplasm（>95%）
- 好發 > 65 歲，男 > 女
- 特徵：
    - 周邊血：成熟小淋巴球增多、**smudge cells（塗抹細胞）**
    - Flow cytometry：CD5（+）（T cell marker 表現於 B cell！）、CD19、CD20（dim）、CD23（+）
    - **與 Mantle cell lymphoma 鑑別**：MCL 也是 CD5(+) B cell，但 MCL → CD23(–)、cyclin D1(+)、t(11;14)
- 分期：Rai staging（0–IV）或 Binet staging（A–C）
- 併發症：
    - **AIHA**（warm type）、ITP → **Evans syndrome**（AIHA + ITP）
    - Hypogammaglobulinemia → 反覆感染
    - **Richter transformation**：轉為 **DLBCL**（大 B 細胞淋巴瘤）→ 預後差
- 預後指標：
    - 良好：del(13q)（最常見）、mutated IGHV
    - 不良：del(17p)（TP53）、del(11q)、unmutated IGHV、ZAP-70(+)、CD38(+)
- 治療：早期（Rai 0–I）→ 觀察（watch and wait）
    - 需治療：進行性淋巴增大、有症狀、Rai III–IV
    - **BTK inhibitor（Ibrutinib）** → 首選
    - Venetoclax（BCL-2 inhibitor）
    - 年輕/fit：FCR（Fludarabine + Cyclophosphamide + Rituximab）
    - del(17p)/TP53 → Ibrutinib 或 Venetoclax 優先

### 3.3 Hairy Cell Leukemia

- 罕見 B cell neoplasm，中年男性
- 特徵：脾腫大（常巨大）、全血球低下、**不常有淋巴結腫大**
- 骨髓：乾抽（dry tap）→ fibrosis
- 細胞：**TRAP（tartrate-resistant acid phosphatase）(+)**；毛狀突起
- Flow cytometry：CD11c, CD25, CD103, CD123
- **BRAF V600E mutation**（幾乎 100%）
- 治療：**Cladribine（2-CdA）**→ 單一療程可達長期緩解（>90%）
- 替代：Pentostatin

## 四、淋巴瘤（Lymphoma）

### 4.1 Hodgkin Lymphoma（HL）

- **雙峰分布**：15–35 歲、>55 歲
- 特徵性病理：**Reed-Sternberg（RS）cells** → 雙核，owl-eye 外觀
    - RS cell markers：**CD15(+)、CD30(+)**
    - RS cell 來源：B cell
- EBV 相關（50% 的 classical HL）
- **Rye 分型**：

| 亞型 | 特點 | 預後 |
|------|------|------|
| **Nodular sclerosis** | **最常見**（70%）、年輕女性、前縱膈腔、collagen bands | 好 |
| Mixed cellularity | 第二常見、EBV 關聯高 | 中 |
| Lymphocyte-rich | RS 少、淋巴球多 | 最好 |
| **Lymphocyte-depleted** | RS 多、淋巴球少、老年/HIV | **最差** |

- Nodular lymphocyte-predominant HL（NLPHL）：**L&H cells（popcorn cells）**、CD20(+)、CD15(–)、CD30(–) → 與 classical HL 不同
- 臨床：
    - 無痛性淋巴結腫大（頸部最常見）
    - **B symptoms**：發燒 >38°C、夜間盜汗、體重減輕 >10%（6 個月內）→ 分期影響
    - **Pel-Ebstein fever**：週期性發燒
    - **酒精誘發淋巴結疼痛**（罕見但具特異性）
    - 擴散方式：**沿相鄰淋巴結順序擴散（contiguous spread）**
- 分期（Ann Arbor Staging）：

| Stage | 定義 |
|-------|------|
| I | 單一淋巴區域 |
| II | 橫膈膜同側 ≥ 2 個淋巴區域 |
| III | 橫膈膜兩側 |
| IV | 肝臟、骨髓或其他結外器官廣泛侵犯 |
| A/B | 無/有 B symptoms |

- 治療：
    - Early stage（I–II）favorable：**ABVD × 2–4 cycles + involved-field RT**
    - Advanced stage（III–IV）：**ABVD × 6 cycles**
        - A = Adriamycin（doxorubicin）→ **心臟毒性**
        - B = Bleomycin → **肺纖維化**
        - V = Vinblastine → 神經毒性
        - D = Dacarbazine
    - 替代：BV-AVD（Brentuximab vedotin + AVD），BEACOPP（較毒但可能更有效於高風險）
    - Relapsed/refractory → **high-dose chemo + auto-HSCT**；Brentuximab vedotin（anti-CD30）；Checkpoint inhibitors（Nivolumab/Pembrolizumab）
- 預後：治癒率 **> 80%**（overall）

### 4.2 Non-Hodgkin Lymphoma（NHL）

- 比 HL 更常見
- 擴散方式：**非連續性（non-contiguous）**
- 分類依侵略性：

#### 低度惡性（Indolent）

**Follicular Lymphoma（FL）**
- **最常見的 indolent NHL**
- t(14;18) → **BCL-2 過度表現**（抗凋亡）
- 好發中老年，淋巴結腫大，進展緩慢
- 可轉為 DLBCL（transformation，30–40%）
- 治療：早期 → RT 或觀察；進行期 → Rituximab ± chemo（R-BENDAMUSTINE、R-CVP）
    - 通常不可治癒但存活長

**Marginal Zone Lymphoma（MZL）**
- **MALT lymphoma**（最常見亞型）：與慢性感染/自體免疫相關
    - 胃 MALT → **Helicobacter pylori**（早期以抗生素三合療法治療可緩解！）
    - 其他部位：唾腺（Sjögren's）、甲狀腺（Hashimoto's）、眼附屬器

**Small Lymphocytic Lymphoma（SLL）**
- CLL 的組織型態，同一疾病譜

#### 高度惡性（Aggressive）

**Diffuse Large B-Cell Lymphoma（DLBCL）**
- **最常見的 NHL**（30–40%）
- 好發 > 60 歲，快速增大的淋巴結
- 治療：**R-CHOP**（Rituximab + Cyclophosphamide + doxorubicin[H] + vincristine[O] + Prednisone）
- 治癒率約 **60–70%**

**Burkitt Lymphoma**
- 高度侵略性，doubling time 極短
- **t(8;14)** → **c-MYC** 過度表現
- **Starry sky pattern**（tingible body macrophages 吞噬凋亡細胞）
- 三種臨床型：
    - **Endemic（非洲型）**：EBV 100%相關 → **下顎/顏面骨腫塊**
    - **Sporadic（散發型）**：腹腔腫塊（迴盲部最常見）
    - **Immunodeficiency-related**：HIV
- 治療：高強度化療（如 R-CODOX-M/IVAC）→ 治癒率高（60–90%）
- **Tumor lysis syndrome（TLS）** 風險極高 → 需預防

**Mantle Cell Lymphoma（MCL）**
- t(11;14) → **Cyclin D1** 過度表現
- CD5(+)、CD23(–)（與 CLL 鑑別）
- 預後較差（中位存活 3–5 年）
- 治療：R-hyperCVAD 或 R-DHAP → auto-HSCT；Ibrutinib 用於 relapse

#### T cell Lymphoma

**Mycosis Fungoides / Sézary Syndrome**
- **皮膚 T 細胞淋巴瘤（CTCL）**
- Mycosis fungoides：patch → plaque → tumor → systemic
    - 病理：Pautrier microabscesses（表皮內 T 細胞浸潤）
- Sézary syndrome：紅皮症（erythroderma）+ **Sézary cells（腦迴狀核 T cell）** 在周邊血中
- CD4(+) T cell

**Adult T-Cell Leukemia/Lymphoma（ATLL）**
- **HTLV-1** 相關
- 表現：皮膚、淋巴結、高血鈣（PTHrP）、flower cells
- 預後極差

### 4.3 Tumor Lysis Syndrome（TLS）

- 大量腫瘤細胞快速溶解 → 釋放細胞內容物
- 高風險：**Burkitt lymphoma、ALL（尤其高白血球）**
- 實驗室：**hyperuricemia、hyperkalemia、hyperphosphatemia、hypocalcemia（Ca 被 PO₄ 結合）**、LDH↑
- 併發症：急性腎衰竭（uric acid / calcium phosphate 沉澱）、心律不整
- 預防：
    - **充分水分補充**（hydration）
    - **Allopurinol**（xanthine oxidase inhibitor）→ 預防用
    - **Rasburicase**（recombinant urate oxidase）→ 已發生 TLS 或極高風險用 → 快速分解 uric acid
    - 注意：Rasburicase 禁用於 G6PD 缺乏（產生 H₂O₂ → 溶血）
- 不建議 urine alkalinization（雖可溶解 uric acid，但增加 calcium phosphate 沉澱）

## 五、漿細胞疾病（Plasma Cell Neoplasms）

### 5.1 Multiple Myeloma（多發性骨髓瘤）

- **惡性漿細胞**增殖於骨髓 → 產生單株免疫球蛋白（M protein）
- 好發 **> 65 歲**，黑人 > 白人

- 臨床表現 **CRAB**：
    - **C**alcium↑（高血鈣）→ 骨溶解釋放
    - **R**enal insufficiency → light chain 沉積、hypercalcemia、uric acid
    - **A**nemia → 骨髓浸潤
    - **B**one lesions → **osteolytic（溶骨性）** → 骨痛、病理性骨折、脊椎壓迫性骨折
- 重要特點：
    - **Bone scan（骨掃描）** 通常 **不會顯示**（因為是溶骨性，無新骨形成 → osteoblast 不活化 → no uptake）→ 用 **skeletal survey（全身骨骼X光）** 或 **low-dose whole-body CT / PET-CT**
    - **Rouleaux formation**（紅血球疊串如銅板）→ ESR 極高
    - M protein 造成假性低血鈉（pseudohyponatremia）
    - 感染（最常見死因）：正常免疫球蛋白減少 → **Strep. pneumoniae** 感染最常見

- 診斷：
    - 血清蛋白電泳（SPEP）：**M spike**（monoclonal protein）
    - 免疫固定電泳（IFE）：確認 M protein 類型
    - **IgG > IgA > IgD > IgE**（依頻率排序）
    - 尿液：**Bence Jones protein（BJP）**= free light chain（κ 或 λ）
    - 骨髓：clonal plasma cells **≥ 10%**（或 biopsy-proven plasmacytoma）
    - Free light chain ratio 異常

- 分期：ISS（International Staging System）：

| ISS Stage | β₂-microglobulin | Albumin |
|-----------|-------------------|---------|
| I | < 3.5 | ≥ 3.5 |
| II | 介於 I 和 III 之間 | — |
| III | ≥ 5.5 | — |

- 治療：
    - 適合移植者（< 70 歲，fit）：**Induction（VRd：Bortezomib + Lenalidomide + Dexamethasone）→ Auto-HSCT → Maintenance（Lenalidomide）**
    - 不適合移植者：VRd 或 DRd（Daratumumab + Rd）
    - 骨病變：**Bisphosphonates（Zoledronic acid）** → 注意 ONJ（osteonecrosis of jaw）
    - 不可治癒但可長期控制

### 5.2 Waldenström Macroglobulinemia（WM）

- 淋巴漿細胞淋巴瘤（LPL）分泌 **IgM** → 高黏滯度
- **MYD88 L265P mutation**（>90%）
- 表現：
    - **Hyperviscosity syndrome**：視力模糊、頭痛、出血、意識變化
    - 貧血、淋巴結/肝脾腫大
    - IgM 可引起冷凝球蛋白血症（cryoglobulinemia）、cold agglutinin disease
- 與 MM 差異：WM 無溶骨病變、無高血鈣
- 治療：Plasmapheresis（緊急降低黏滯度）、Rituximab-based chemo、Ibrutinib
- **禁忌使用 Rituximab flare** → 先行 plasmapheresis 再給

### 5.3 MGUS（Monoclonal Gammopathy of Undetermined Significance）

- M protein < 3 g/dL、骨髓漿細胞 < 10%、無 CRAB
- 進展至 MM 風險：每年 **約 1%**
- 需長期追蹤

### 5.4 Primary Amyloidosis（AL Amyloidosis）

- Light chain 沉積為不溶性類澱粉纖維
- 器官受影響：心臟（**restrictive cardiomyopathy**，最常見死因）、腎臟（nephrotic syndrome）、肝臟、舌（macroglossia）、神經、皮膚（periorbital purpura, waxy skin）
- 診斷：組織切片 **Congo red stain → apple-green birefringence under polarized light**
- 治療：類似 MM（Bortezomib-based + auto-HSCT for eligible patients）

## 六、骨髓增生性腫瘤（Myeloproliferative Neoplasms, MPN）

### 6.1 總論

- 造血幹細胞 clonal disorder → 一或多系骨髓細胞過度增殖
- 共通特徵：脾腫大、骨髓過度增生、血栓/出血風險↑
- 可相互轉化，皆可進展至骨髓纖維化或 AML

### 6.2 Polycythemia Vera（PV, 真性紅血球增多症）

- **JAK2 V617F mutation**（>95%）
- 紅血球過度製造（也可有白血球、血小板↑）
- 表現：
    - **Plethora（面紅）**、頭痛、頭暈、視力模糊
    - **水生性搔癢（aquagenic pruritus）**→ 洗澡後搔癢（與嗜鹼性球/mast cell 釋放 histamine 相關）
    - **紅斑性肢痛症（erythromelalgia）**→ 手腳灼熱/紅腫/痛（與血小板微血管栓塞相關）→ 對 aspirin 反應佳
    - 血栓（最主要死因：心肌梗塞、中風、Budd-Chiari syndrome）
    - 痛風（uric acid↑）
    - **Budd-Chiari syndrome**（肝靜脈血栓）→ 年輕人不明原因 Budd-Chiari → 想到 PV/PNH
    - 消化性潰瘍（histamine↑）
    - 脾腫大
- 實驗室：Hb/Hct↑、WBC↑、Plt↑、**EPO↓**（與次發性紅血球增多症不同，後者 EPO↑）
    - **LAP score↑**（與 CML 區別）
- 診斷標準（WHO）：Hb > 16.5（男）/ 16（女）或 Hct > 49/48% + JAK2 V617F + 骨髓（hypercellular with trilineage proliferation）
- 治療：
    - 所有人：**Phlebotomy（放血）**（目標 Hct < 45%）+ **低劑量 Aspirin**
    - 高風險（>60 歲或有血栓史）→ 加 **Hydroxyurea**（cytoreduction 首選）
- 自然病程：可進展為 post-PV myelofibrosis（spent phase）或 AML

### 6.3 Essential Thrombocythemia（ET, 原發性血小板增多症）

- **JAK2 V617F**（50–60%）、**CALR mutation**（25–30%）、MPL mutation（5%）
- 血小板 > 450,000/μL（持續性）
- 表現：血栓（動脈 > 靜脈）、erythromelalgia；極高血小板數（>1,000K）反而可因 acquired vWD → **出血**
- 排除反應性血小板增多（感染、缺鐵、術後、脾切除後）
- 治療：
    - 低風險：低劑量 Aspirin ± 觀察
    - 高風險：Hydroxyurea + Aspirin
    - Anagrelide（降血小板特定藥物）
- 預後最好的 MPN

### 6.4 Primary Myelofibrosis（PMF, 原發性骨髓纖維化）

- **JAK2 V617F**（50–60%）、CALR（25–35%）、MPL
- 骨髓纖維化 → 造血衰竭 → 髓外造血
- 骨髓切片：**reticulin/collagen fibrosis** → dry tap（抽不到骨髓）
- 血抹片：**leukoerythroblastic picture**（nucleated RBC + immature WBC in peripheral blood）、**tear-drop cells（dacrocytes/淚滴狀紅血球）**
- 表現：**巨脾（最顯著特徵）**、貧血、全身症狀（體重減輕、盜汗、疲倦）
- 治療：
    - **Ruxolitinib**（JAK1/2 inhibitor）→ 縮小脾臟、改善症狀
    - 貧血：EPO、輸血、Danazol
    - **Allo-HSCT**：唯一根治（高風險病人考慮）
    - 脾切除：嚴重脾亢或症狀性巨脾（但併發症高）
- 預後：最差的 MPN（除 CML blast crisis 外），中位存活 3–7 年

### 6.5 MPN 鑑別摘要

| 疾病 | 主要增高 | JAK2 V617F | 特殊特徵 |
|------|----------|------------|----------|
| PV | RBC | >95% | EPO↓、aquagenic pruritus |
| ET | Platelets | 50–60% | Erythromelalgia、出血/血栓 |
| PMF | Fibrosis | 50–60% | Tear-drop cells、巨脾、dry tap |
| CML | WBC | 無 | BCR-ABL(+)、Ph chromosome、LAP↓ |

## 七、骨髓分化不良症候群（Myelodysplastic Syndromes, MDS）

### 7.1 總論

- 造血幹細胞 clonal disorder → 無效造血（ineffective hematopoiesis）→ 周邊血球低下但骨髓增生
- 好發 > 65 歲
- **癌前狀態**：約 30% 進展為 AML
- 原因：原發性（de novo）最常見；次發性（prior chemo/RT → **alkylating agents, topoisomerase II inhibitors**）

### 7.2 臨床表現

- 慢性血球低下：疲倦（貧血）、感染（嗜中性球↓）、出血（血小板↓）
- **大球性貧血（macrocytic anemia）** 為最常見的初始表現
- 常為偶然發現的 CBC 異常

### 7.3 診斷

- 骨髓切片：**異常型態（dysplasia）**影響 ≥ 1 系 + 排除其他原因
    - 紅血球系：ring sideroblasts（Prussian blue stain）、核芽異常
    - 白血球系：hyposegmented/pseudo-Pelger-Huët cells、hypogranular neutrophils
    - 巨核細胞系：micro-megakaryocytes
- 周邊血：macrocytosis、pancytopenia
- **Blasts < 20%**（≥ 20% 即為 AML）

### 7.4 細胞遺傳學與預後

| 染色體異常 | 預後 |
|------------|------|
| del(5q)（isolated） | **良好**（5q– syndrome：貧血 + normal/↑ plt + hypolobated megakaryocytes → Lenalidomide 有效） |
| Normal karyotype | 中等 |
| del(7) / complex karyotype | **不良** |

- **IPSS-R**（Revised International Prognostic Scoring System）：根據血球低下嚴重度、blast 比例、細胞遺傳學分組

### 7.5 治療

- 低風險（IPSS-R lower risk）：
    - 支持療法（輸血 + EPO ± G-CSF）
    - del(5q) → **Lenalidomide**（high response rate）
    - Luspatercept（用於 ring sideroblasts 亞型）
- 高風險（IPSS-R higher risk）：
    - **Azacitidine 或 Decitabine**（hypomethylating agents, HMA）→ 首選
    - Allo-HSCT：唯一根治（年輕/fit 高風險者）

## 八、出血與凝血疾病

### 8.1 出血型態與定位

| 特徵 | 血小板/血管異常（初級止血） | 凝血因子異常（次級止血） |
|------|---------------------------|------------------------|
| 出血部位 | 皮膚黏膜（瘀斑、紫斑、鼻出血、牙齦出血、月經過多） | 深層組織（關節、肌肉、後腹腔） |
| 出血時間 | 立即 | 延遲 |
| 瘀青型態 | 小點狀（petechiae） | 大片（ecchymosis、hematoma） |
| 手術後出血 | 即刻 | 數小時後 |

### 8.2 血小板低下（Thrombocytopenia）

#### 免疫性血小板低下紫斑症（Immune Thrombocytopenia, ITP）

- 抗血小板抗體（anti-GPIIb/IIIa Ab）→ 脾臟破壞血小板
- 分類：
    - 兒童：常在病毒感染後急性發作，多**自限性**
    - 成人：通常慢性，女性多
- 實驗室：**isolated thrombocytopenia**（其他血球正常）、骨髓：megakaryocyte↑、**排除診斷**（無特異性檢查）
- 周邊血抹片：大血小板（young platelets）
- 治療：
    - Plt > 30K 且無出血 → 觀察
    - 首選：**Corticosteroids**（Prednisone）
    - 二線：**IVIG**（快速升血小板，用於急性出血或術前）、Anti-D immunoglobulin（Rh+ 且有脾的患者）
    - 慢性/難治：**Splenectomy**、**Rituximab**、**TPO-receptor agonists（Eltrombopag, Romiplostim）**
    - **Platelet transfusion**：ITP 通常不輸，除非危及生命出血

#### 血栓性血小板低下紫斑症（Thrombotic Thrombocytopenic Purpura, TTP）

- **ADAMTS13 缺乏**（先天或後天自體抗體）→ ultra-large vWF multimers 無法被切割 → 微血管血栓 → 血小板消耗 + MAHA
- 經典五聯徵（pentad）：
    1. **Thrombocytopenia**
    2. **MAHA**（schistocytes）
    3. **神經症狀**（confusion、seizure、focal deficit）
    4. **腎功能異常**
    5. **發燒**
- 實驗室：MAHA（schistocytes、LDH↑、haptoglobin↓、indirect bilirubin↑）+ 血小板↓；**PT/aPTT 正常**（與 DIC 不同！）、ADAMTS13 activity < 10%
- 治療：**Plasma exchange（TPE / plasmapheresis）** → 最重要且最先做
    - Steroids
    - Caplacizumab（anti-vWF）
    - **禁忌輸血小板**（加重血栓！）
    - 不治療死亡率 > 90%

#### 溶血性尿毒症症候群（Hemolytic Uremic Syndrome, HUS）

- 與 TTP 類似（MAHA + thrombocytopenia）但**以腎衰竭為主**
- 典型（D+ HUS）：兒童，**E. coli O157:H7**（EHEC）→ Shiga-like toxin → 腎小血管內皮損傷
    - 前驅：血性腹瀉
    - **不可使用抗生素**（加速毒素釋放）
    - 治療：支持療法（輸液 + 透析）
- 非典型（aHUS）：complement-mediated → **Eculizumab**（anti-C5）

#### DIC（Disseminated Intravascular Coagulation）

- 全身性凝血活化 + 纖溶 → 消耗性凝血病
- 原因：sepsis（最常見）、trauma、obstetric（羊水栓塞、胎盤早期剝離）、AML-M3、malignancy
- 實驗室：PT↑、aPTT↑、**fibrinogen↓**、**D-dimer↑**、platelet↓、schistocytes
- 治療：**治療根本原因**（最重要）→ 補充血品（FFP、cryoprecipitate if fibrinogen < 100、platelets）

### 8.3 血小板功能異常

- **Bernard-Soulier syndrome**：GPIb 缺乏 → 不能與 vWF 黏附；**giant platelets**、AR
- **Glanzmann thrombasthenia**：GPIIb/IIIa 缺乏 → 不能聚集；AR
- **Uremia** → 血小板功能異常 → 治療：DDAVP（Desmopressin）

### 8.4 凝血因子異常

#### von Willebrand Disease（vWD）

- **最常見的遺傳性出血疾病**
- vWF 功能：攜帶 Factor VIII + 黏附血小板至內皮（via GPIb）
- 分型：

| 類型 | 遺傳 | vWF | VIII | 特徵 |
|------|------|-----|------|------|
| Type 1 | AD | ↓（quantitative） | ↓ | **最常見**（70–80%），輕度 |
| Type 2 | AD | 質異常（qualitative） | ↓ 或 N | 多亞型（2A, 2B, 2M, 2N） |
| Type 3 | AR | **極低/缺乏** | ↓↓ | 最嚴重 |

- 出血型態：**黏膜出血**（鼻出血、月經過多、術後出血）
- 實驗室：aPTT↑ 或正常、BT↑（PFA-100↑）、**ristocetin cofactor activity↓**
    - Type 2B：ristocetin-induced platelet aggregation（RIPA）增強（低濃度即聚集）→ 血小板↓
- 治療：
    - Type 1：**DDAVP（Desmopressin）**（促進內皮釋放 vWF）→ 首選
    - Type 2/3 或重度出血：vWF-containing FVIII concentrates（Humate-P）
    - **Type 2B → 禁用 DDAVP**（加重血小板低下）
    - Aminocaproic acid / Tranexamic acid（抗纖溶，黏膜出血輔助用）

#### 血友病（Hemophilia）

| 疾病 | 缺乏 | 遺傳 | 實驗室 |
|------|------|------|--------|
| Hemophilia A | **Factor VIII** | **X-linked recessive** | aPTT↑、PT 正常、BT 正常 |
| Hemophilia B（Christmas disease） | **Factor IX** | X-linked recessive | aPTT↑、PT 正常 |

- 嚴重度依因子活性：<1% 重度、1–5% 中度、5–40% 輕度
- 表現：深層出血（**hemarthrosis 最常見表現**，膝 > 肘 > 踝）、肌肉血腫、顱內出血
- 治療：
    - Hemophilia A：Factor VIII concentrate（recombinant）、DDAVP（輕度）
    - Hemophilia B：Factor IX concentrate
    - **Emicizumab**（bispecific Ab mimicking Factor VIII）→ Hemophilia A 預防用
    - 抑制抗體（inhibitors）發展時：bypassing agents（FEIBA、rFVIIa）
- **不可使用 Aspirin / IM injection**

### 8.5 凝血路徑與實驗室監測摘要

| 路徑 | 監測 | 涉及因子 |
|------|------|----------|
| Extrinsic | **PT / INR** | VII（+ common） |
| Intrinsic | **aPTT** | XII, XI, IX, VIII（+ common） |
| Common | PT + aPTT | X, V, II, fibrinogen, XIII |
| Fibrinolysis | D-dimer、FDP | — |

- **Mixing study**：aPTT 延長時，混合正常血漿
    - 若校正 → 因子缺乏
    - 若不校正 → **抑制物（inhibitor）**（如 lupus anticoagulant、Factor VIII inhibitor）

## 九、血栓性疾病（Thrombophilia）

### 9.1 遺傳性血栓傾向

| 疾病 | 機制 | 備註 |
|------|------|------|
| **Factor V Leiden** | FV 對 Activated Protein C（APC）抵抗 | **最常見的遺傳性血栓因素**（白種人） |
| **Prothrombin G20210A mutation** | Prothrombin 增加 | 第二常見 |
| Protein C 缺乏 | 抗凝蛋白↓ | **Warfarin-induced skin necrosis**（Warfarin 先降 Protein C → 短暫高凝狀態） |
| Protein S 缺乏 | 抗凝蛋白↓ | — |
| Antithrombin III 缺乏 | Heparin 效果↓ | **Heparin resistance**（heparin 需 ATIII 才能作用） |

### 9.2 後天性血栓傾向

- **Antiphospholipid syndrome（APS）**：
    - 抗體：lupus anticoagulant（LA）、anticardiolipin Ab、anti-β₂-glycoprotein I Ab
    - 矛盾：aPTT **延長**（in vitro 干擾磷脂）但 **in vivo → 血栓**
    - 表現：動靜脈血栓、反覆流產、thrombocytopenia
    - 與 SLE 密切相關
    - 治療：長期 **Warfarin**（INR 2–3）

### 9.3 Heparin-Induced Thrombocytopenia（HIT）

- Type II：heparin 使用 5–10 天後 platelet↓ >50%
- 機制：Anti-PF4/heparin complex Ab → 血小板活化 → **血栓**（非出血！）
- 診斷：4T score（Timing、Thrombocytopenia、Thrombosis、oTher causes）+ anti-PF4 Ab（ELISA）+ serotonin release assay（gold standard）
- 處理：**立即停所有 heparin**（包括 heparin flushes）→ 替代抗凝血：**Argatroban**（肝代謝）或 Bivalirudin（direct thrombin inhibitors）
- **不可用 Warfarin**（急性期會加重 → Protein C↓ → venous limb gangrene）→ 待血小板恢復後才能過渡至 Warfarin
- **不可輸血小板**（加重血栓）

## 十、輸血醫學

### 10.1 血型系統

- **ABO 系統**：

| 血型 | RBC 表面抗原 | 血清抗體 |
|------|-------------|----------|
| A | A | Anti-B（IgM） |
| B | B | Anti-A（IgM） |
| AB | A + B | 無 → **universal recipient（全適受者）** |
| O | 無（H antigen） | Anti-A + Anti-B → **universal donor（全適供者）** |

- **Rh 系統**：D 抗原最重要
    - Rh(–) 母親懷 Rh(+) 胎兒 → 致敏 → 第二胎 **新生兒溶血症（HDN）**
    - 預防：**RhoGAM（Anti-D immunoglobulin）**，28 週 + 72 小時內產後給予

- **Bombay phenotype（Oh）**：缺乏 H antigen → 無法合成 A/B 抗原 → 只能接受 Bombay 型的血

### 10.2 血品特性與使用

| 血品 | 內容 | 適應症 | 備註 |
|------|------|--------|------|
| Packed RBC | 紅血球（Hct ~55–65%） | Hb < 7（一般）或 < 10（心臟病/症狀性） | 1 unit 升 Hb ~1 g/dL |
| FFP | 所有凝血因子 | PT/aPTT > 1.5×正常且出血、liver disease、warfarin reversal（急性，但 PCC 更優先）、DIC | ABO 需相容 |
| Platelets | 血小板 | < 10K（預防）、< 50K（手術/侵入性處置）、< 100K（神經外科/眼科手術） | 1 unit 升 ~5,000–10,000/μL |
| Cryoprecipitate | Fibrinogen、FVIII、XIII、vWF | Fibrinogen < 100–150、DIC、vWD（無法取得 Humate-P 時） | — |

### 10.3 輸血反應

| 反應 | 時間 | 機制 | 處理 |
|------|------|------|------|
| **急性溶血反應** | 分鐘–小時 | **ABO 不合**（最嚴重）；IgM + complement → intravascular hemolysis | **立即停血**、NS 輸液、維持尿量、支持 |
| **發燒性非溶血反應** | 1–6 hr | **最常見**；白血球 cytokines | Acetaminophen；leukoreduced blood 預防 |
| **過敏反應** | 分鐘 | IgE 媒介；**IgA 缺乏者最嚴重**（anti-IgA → anaphylaxis） | Antihistamine；IgA 缺乏 → **washed RBC** |
| **TRALI** | < 6 hr | Donor anti-leukocyte Ab → 肺損傷 | 支持療法（O₂、必要時 intubation）；非心因性肺水腫 |
| **TACO** | 小時 | 容量超載 → 心因性肺水腫 | Diuretics（與 TRALI 鑑別！） |
| **延遲性溶血反應** | 2–10 天 | Anamnestic Ab response → extravascular hemolysis | 通常輕微 |

- **TRALI vs TACO**：

| 特徵 | TRALI | TACO |
|------|-------|------|
| BNP | 正常 | ↑ |
| PCWP | 正常 | ↑ |
| 機制 | 肺損傷 | 容量超載 |
| 體溫 | 可發燒 | 通常不發燒 |
| 治療 | 支持 | 利尿劑 |

### 10.4 特殊處理血品

- **Leukoreduced（白血球過濾）**：預防發燒性非溶血反應、CMV 傳播
- **Irradiated（照射）**：預防 **TA-GVHD**（transfusion-associated graft-versus-host disease）→ 用於免疫缺損病人、directed donation from relatives、BMT、Hodgkin lymphoma
- **Washed RBC**：去除血漿蛋白 → IgA 缺乏者
- **CMV-negative**：用於 CMV seronegative 移植受者（或用 leukoreduced 替代）

## 十一、腫瘤學總論（Oncology Principles）

### 11.1 腫瘤生物學

- **Hallmarks of cancer**（Hanahan & Weinberg）：
    - 維持增殖信號、逃避生長抑制、抵抗細胞死亡、啟動複製永生化、誘導血管新生、活化侵襲與轉移、能量代謝改變、逃避免疫破壞、基因組不穩定、腫瘤促進性發炎

- **癌基因（Oncogene）** vs **抑癌基因（Tumor suppressor gene）**：

| | 癌基因 | 抑癌基因 |
|--|--------|----------|
| 機制 | Gain-of-function | Loss-of-function |
| 遺傳方式 | **顯性**（1 個 allele 突變即致癌） | **隱性**（Knudson's two-hit hypothesis：需 2 個 allele 皆失活） |
| 例子 | RAS、MYC、HER2、BCR-ABL、RET | RB、TP53、APC、BRCA1/2、VHL |

- 重要癌基因：

| 癌基因 | 腫瘤 |
|--------|------|
| HER2/neu（ERBB2） | 乳癌、胃癌 |
| RAS（KRAS） | 大腸癌、胰臟癌、肺癌 |
| MYC | Burkitt lymphoma |
| BCR-ABL | CML、Ph(+) ALL |
| RET | MEN2、甲狀腺髓質癌 |
| BRAF | 黑色素瘤、Hairy cell leukemia |
| ALK | 非小細胞肺癌（adenocarcinoma） |
| KIT（c-KIT） | GIST |

- 重要抑癌基因：

| 抑癌基因 | 腫瘤 |
|----------|------|
| **TP53** | **最常見突變的基因**；Li-Fraumeni syndrome（AD → 多種癌）|
| RB | 視網膜母細胞瘤 |
| APC | FAP（familial adenomatous polyposis）→ 大腸癌 |
| BRCA1/2 | 乳癌、卵巢癌 |
| VHL | von Hippel-Lindau → RCC（clear cell）、pheochromocytoma、hemangioblastoma |
| WT1 | Wilms tumor |
| NF1 | Neurofibromatosis type 1 |
| PTEN | Cowden syndrome → breast、thyroid、endometrial |

### 11.4 腫瘤急症（Oncologic Emergencies）

#### Superior Vena Cava（SVC）Syndrome

- 原因：**肺癌（最常見，尤其 SCLC）**、淋巴瘤、mediastinal mass
- 症狀：顏面/頸部/上肢水腫、頸靜脈怒張、dyspnea、collateral veins on chest
- 診斷：CT with contrast
- 治療：取決於原因 → SCLC/lymphoma → 化療/RT；NSCLC → RT ± stenting；SVC stent（快速緩解）

#### Spinal Cord Compression

- 原因：轉移性腫瘤（**乳癌、肺癌、攝護腺癌**最常見）→ 脊椎硬膜外壓迫
- 症狀：back pain（最早且最常見）→ 下肢無力 → 感覺異常 → 大小便失禁
- 診斷：**MRI（whole spine）**
- 治療：**Dexamethasone（高劑量，立即給予）** → 手術減壓 ± RT，或單獨 RT

#### Febrile Neutropenia

- 定義：ANC < 500（或預期 48 hr 內 < 500）+ 體溫 ≥ 38.3°C（一次）或 ≥ 38.0°C（持續 1 hr）
- 最常見感染源：skin/soft tissue、lung、GI tract
- 最常見致病菌：**Gram-negative bacilli**（E. coli, Klebsiella, Pseudomonas）；**Gram-positive 增加中**（尤其 line-related）
- 處理：
    - 立即**血液培養 × 2 sets**（包括 central line 抽血）+ 經驗性廣效抗生素
    - **Anti-pseudomonal β-lactam**：Cefepime、Piperacillin-tazobactam、Meropenem（擇一）
    - 加 Vancomycin 時機：mucositis、skin/catheter 感染疑慮、blood culture Gram(+)、hemodynamic instability
    - **G-CSF** 可考慮（非常規）
    - 不明熱持續 4–7 天 → 加 empiric **antifungal（Caspofungin 或 Voriconazole）**

#### Hypercalcemia of Malignancy

- 機制：PTHrP（最常見）、骨轉移溶骨、1,25-D₃（lymphoma）
- 處理：**大量 NS 輸液** → **Zoledronic acid（bisphosphonate）** → Calcitonin（快速但短效）→ Denosumab（bisphosphonate 無效時）

### 11.5 化療原則與毒性

#### 細胞週期

- **G0**：靜止期
- **G1**：RNA/protein 合成 → 準備 DNA 合成
- **S**：DNA 合成
- **G2**：準備有絲分裂
- **M**：有絲分裂

- **Cell cycle-specific agents**：作用於特定期（S phase：Methotrexate、6-MP、Cytarabine、5-FU；M phase：Vinca alkaloids、Taxanes）→ 對快速分裂細胞效果好
- **Cell cycle-nonspecific agents**：任何期皆有效（Alkylating agents、Platinum、Anthracyclines）

#### 重要化療藥物毒性

| 藥物 | 特殊毒性 |
|------|----------|
| **Doxorubicin（Adriamycin）** | **心臟毒性（dilated cardiomyopathy）**→ 與累積劑量相關，Dexrazoxane 預防 |
| **Bleomycin** | **肺纖維化**（dose-related） |
| **Cisplatin** | 腎毒性（需 hydration）、耳毒性、周邊神經病變、嚴重噁心 |
| **Cyclophosphamide / Ifosfamide** | **出血性膀胱炎**（acrolein 代謝物）→ 預防：**MESNA** |
| **Vincristine** | 周邊神經病變（dose-limiting）、便秘（autonomic） |
| **Methotrexate** | 口腔黏膜炎、骨髓抑制、肝毒性 → 解毒：**Leucovorin（folinic acid）rescue** |
| **5-FU** | 骨髓抑制、mucositis、hand-foot syndrome；DPD 缺乏者毒性極嚴重 |
| **Taxanes（Paclitaxel, Docetaxel）** | 周邊神經病變、骨髓抑制 |
| **Irinotecan** | 腹瀉（早期膽鹼性 + 晚期） |
| **Bortezomib** | 周邊神經病變、blood小板↓ |
| **Thalidomide / Lenalidomide** | DVT/PE、致畸性 |
| **Rituximab** | Infusion reaction、Hepatitis B reactivation、PML |
| **Bevacizumab** | 高血壓、蛋白尿、傷口癒合不良、穿孔、出血 |
| **Trastuzumab** | 心臟毒性（不與 Doxorubicin 同時用） |
| **Immune checkpoint inhibitors（anti-PD-1/PD-L1/CTLA-4）** | 免疫相關副作用（irAE）：colitis、pneumonitis、hepatitis、thyroiditis、hypophysitis、skin rash |

### 11.6 標靶治療與免疫治療

#### 標靶治療

#### 免疫治療

- **Immune checkpoint inhibitors**：
    - Anti-PD-1：Nivolumab、Pembrolizumab
    - Anti-PD-L1：Atezolizumab、Durvalumab
    - Anti-CTLA-4：Ipilimumab
    - 適應症：melanoma、NSCLC、RCC、Hodgkin lymphoma、urothelial cancer、MSI-high tumors 等
- **CAR-T cell therapy**：
    - 自體 T cell 基因改造表達 chimeric antigen receptor → 辨識腫瘤抗原（如 CD19）
    - 適應症：relapsed/refractory B-ALL、DLBCL
    - 主要副作用：**Cytokine Release Syndrome（CRS）** → 發燒、低血壓、呼吸困難 → 治療：**Tocilizumab（anti-IL-6R）**；ICANS（neurotoxicity）

## 十二、常見實體腫瘤血液腫瘤科相關重點

### 12.1 造血幹細胞移植（HSCT）

#### 類型

| 類型 | 來源 | 優點 | 缺點 |
|------|------|------|------|
| **Autologous** | 自體 | 無 GVHD、無需配對 | 無 GVL effect、可能回輸腫瘤細胞 |
| **Allogeneic** | 他人（sibling/MUD/haplo/cord blood） | **GVL（graft-versus-leukemia）effect** | GVHD 風險、需 HLA 配對 |

- **HLA 配對**：最重要為 **HLA-A, B, C, DRB1**（8/8 match）
- 幹細胞來源：骨髓、周邊血（G-CSF mobilized）、臍帶血

#### 適應症

- Auto-HSCT：relapsed Hodgkin/NHL、multiple myeloma（consolidation）
- Allo-HSCT：AML（high-risk）、ALL（high-risk/relapsed）、MDS、CML（TKI failure）、aplastic anemia、thalassemia major

#### 移植物抗宿主疾病（GVHD）

| | 急性 GVHD（< 100 天） | 慢性 GVHD（> 100 天） |
|--|----------------------|----------------------|
| 皮膚 | Maculopapular rash | Scleroderma-like、lichen planus-like |
| 肝 | 黃疸（cholestatic）、肝功能異常 | 同左 + cholestasis |
| GI | 腹瀉（watery/bloody）、腹痛、噁心 | 口腔黏膜炎、食道狹窄、乾燥 |
| 其他 | — | 乾眼症、乾口症、bronchiolitis obliterans |

- 治療：**Corticosteroids**（首選）→ 二線：Ruxolitinib
- 預防：Tacrolimus/Cyclosporine + Methotrexate/Mycophenolate
- **GVL effect（graft-versus-leukemia）**：donor T cells 同時攻擊殘留腫瘤 → 降低復發 → allo-HSCT 的治療核心

#### 移植後感染時序

- 與實體器官移植類似：
    - 前 30 天：細菌/黴菌（neutropenic phase）
    - 30–100 天：CMV（最重要）、PJP、Aspergillus → CMV 監測與預防性治療
    - > 100 天：encapsulated bacteria、VZV reactivation、EBV → PTLD

### 12.2 腫瘤相關凝血異常

- **Trousseau syndrome**：malignancy-associated migratory superficial thrombophlebitis → 特別見於 pancreatic cancer、mucinous adenocarcinoma
- 惡性腫瘤相關 DIC：尤其 APL（M3）、mucin-secreting adenocarcinoma
- Warfarin-resistant DVT → 考慮 LMWH 長期使用（cancer-associated thrombosis）

## 十三、其他重要主題

### 13.1 脾臟

- 功能：過濾老化/異常 RBC、免疫（opsonization of encapsulated bacteria）、造血（胎兒期/病理性）
- **脾切除後風險**：
    - **OPSI（Overwhelming Post-Splenectomy Infection）**：encapsulated bacteria → **Strep. pneumoniae**（最常見）、N. meningitidis、H. influenzae
    - 預防：術前/術後疫苗接種 + 必要時 prophylactic penicillin
- 脾切除後血液學變化：
    - **Howell-Jolly bodies**（核殘餘物）
    - Target cells
    - 暫時性血小板升高（thrombocytosis）
    - Heinz bodies 可見

### 13.3 Methemoglobinemia

- 定義：Fe²⁺（ferrous）被氧化為 Fe³⁺（ferric）→ 不能攜帶 O₂
- 原因：先天（Cytochrome b5 reductase 缺乏）、藥物（**Dapsone**、local anesthetics 如 Benzocaine、Nitrates、Sulfonamides）
- 表現：**SpO₂ ≈ 85%（不因 O₂ 而改善）**、發紺、chocolate-brown blood（不因曝氣而變紅）
- 治療：**Methylene blue**（1–2 mg/kg IV）→ 作為電子載體還原 MetHb
    - G6PD 缺乏者禁用 Methylene blue → 改用 Ascorbic acid

### 13.4 Porphyria

- 血基質（heme）合成途徑酵素缺陷

| 類型 | 缺陷酵素 | 表現 |
|------|----------|------|
| **Acute Intermittent Porphyria（AIP）** | PBG deaminase（HMB synthase）| 腹痛、精神症狀、神經病變、紅色尿；**無皮膚症狀** |
| Porphyria Cutanea Tarda（PCT） | Uroporphyrinogen decarboxylase | **最常見的 porphyria**；光敏感、水泡、多毛、與 HCV/鐵過多/酒精相關 |
| Erythropoietic Protoporphyria | Ferrochelatase | 光敏感、但無水泡 |

- AIP 觸發因子（5P）：drugs（P450 inducers）、fasting、alcohol、infection、hormones（progesterone）
- AIP 治療：**Hemin（heme arginate）IV** + glucose loading + 避免觸發
- AIP 診斷：尿液 **PBG↑↑（porphobilinogen）** + ALA↑
