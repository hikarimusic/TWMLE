# 麻醉科

---

## 一、麻醉前評估與準備

### 1.1 ASA 身體狀態分類（ASA Physical Status Classification）

| 分級 | 定義 | 舉例 |
|------|------|------|
| ASA I | 正常健康 | 健康、非吸菸、無/極少飲酒 |
| ASA II | 輕度全身性疾病 | 控制良好的 DM/HTN、吸菸、BMI 30–40、懷孕 |
| ASA III | 嚴重全身性疾病 | 控制不良 DM/HTN、COPD、BMI ≥40、活動性肝炎、酒精依賴、裝有 pacemaker、ESRD（規律透析）、MI/CVA/TIA >3 個月、EF 降低 |
| ASA IV | 持續威脅生命之嚴重疾病 | 近期 MI/CVA（<3 個月）、嚴重瓣膜疾病、sepsis、DIC、ARDS |
| ASA V | 不手術即將死亡 | 破裂 AAA、massive trauma、顱內出血 with herniation |
| ASA VI | 腦死器官捐贈者 | — |

- 急診加上 **"E"**（Emergency）

### 1.2 Mallampati 分級（評估困難插管）

| 分級 | 可見結構 |
|------|----------|
| I | 軟顎 + 懸雍垂 + 扁桃腺柱 |
| II | 軟顎 + 懸雍垂 |
| III | 軟顎 + 懸雍垂基部 |
| IV | 僅見硬顎 |

- Class III/IV 預示困難插管
- 其他困難插管預測指標：甲頦距離（thyromental distance）< 6 cm（約三指幅）、張口度 < 3 cm（約二指幅）、頸部活動受限、短頸肥胖、上門齒突出（buck teeth）、下顎後縮（retrognathia）

### 1.3 NPO 禁食指引（ASA Fasting Guidelines）

| 攝入物 | 最短禁食時間 |
|--------|------------|
| 清澈液體（水、無渣果汁、黑咖啡、茶） | **2 小時** |
| 母乳 | **4 小時** |
| 嬰兒配方奶、牛奶、輕食 | **6 小時** |
| 油炸/高脂/肉類正餐 | **8 小時**（或更久） |

- 目的：減少 **aspiration pneumonitis（Mendelson's syndrome）** 風險
- 胃內容物 pH < 2.5 且量 > 25 mL → 高風險
- 高誤吸風險族群：未禁食急診病人、肥胖、懷孕、GERD、DM gastroparesis、腸阻塞
- 急診未禁食病人 → **Rapid Sequence Induction/Intubation（RSI）**

### 1.4 術前藥物調整（麻醉相關重點）

- **繼續**：β-blockers（突然停用 → rebound tachycardia/HTN/MI）、抗癲癇藥、甲狀腺藥、類固醇（長期使用者給 stress dose）、吸入型支氣管擴張劑
- **停用**：
    - MAOi：傳統型需術前 2 週停（與 meperidine 交互作用 → serotonin syndrome / hypertensive crisis）
    - Metformin：術日停（lactic acidosis 風險）
    - ACEi/ARB：術日停（術中頑固性低血壓）
    - 抗凝血劑/抗血小板：依手術與出血風險調整（參見外科概論）
- **Aspiration prophylaxis**：
    - Metoclopramide（促胃排空）、H₂ blocker（ranitidine）、PPI、sodium citrate（非顆粒性制酸劑，提高胃 pH）

### 1.5 術前同意與知情告知

- 說明麻醉方式、風險（牙齒損傷、聲帶損傷、神經損傷、惡性高熱、過敏反應、awareness）
- 全身麻醉中 **awareness（術中知曉）**：發生率約 0.1–0.2%，BIS 監測可降低風險

## 二、呼吸道管理

### 2.1 基本呼吸道手法

- **Head tilt–chin lift**：一般病人打開呼吸道的首選手法
- **Jaw thrust**：**懷疑 C-spine 損傷時使用**（不動頸椎）
- 口咽呼吸道（OPA, oropharyngeal airway）：僅用於無意識病人（有意識 → gag reflex → 嘔吐）
- 鼻咽呼吸道（NPA, nasopharyngeal airway）：可用於有意識病人；**禁忌：顱底骨折**

### 2.2 面罩通氣（Bag-Mask Ventilation）

- 困難面罩通氣預測因子（**MOANS** 或 **BONES**）：
    - **M**ask seal 困難（鬍鬚、顏面外傷）
    - **O**besity / Obstruction（肥胖、上呼吸道阻塞）
    - **A**ge > 55
    - **N**o teeth（無牙齒 → 面罩不易密合）
    - **S**tiffness / Snoring（肺順應性差 / 打鼾 = OSA）

### 2.3 喉頭鏡直接插管（Direct Laryngoscopy & Intubation）

- **Cormack-Lehane 分級**：
    - Grade I：聲門完全可見
    - Grade II：僅見聲門後部（杓狀軟骨）
    - Grade III：僅見會厭
    - Grade IV：什麼都看不到（連會厭都看不到）
- Grade III/IV → 困難插管

- **氣管內管（ETT）尺寸**：
    - 成人男性：7.5–8.0 mm ID
    - 成人女性：7.0–7.5 mm ID
    - 小兒（>1 歲）：（年齡/4）+ 4（uncuffed）或（年齡/4）+ 3.5（cuffed）
    - 新生兒：3.0–3.5 mm

- **確認管位**：
    - **最可靠：End-tidal CO₂（ETCO₂/capnography）**
    - 五點聽診法（雙側肺尖、雙側肺底、上腹部）
    - CXR：管端位於 carina 上方 2–4 cm（T3–T4 水平）
    - 右主支氣管插管（endobronchial intubation）最常見（右側較垂直、較粗）→ 左肺呼吸音消失

- **Cuff pressure**：維持 20–30 cmH₂O（太高 → 氣管黏膜缺血壞死；太低 → 漏氣/誤吸）

### 2.4 Rapid Sequence Induction/Intubation（RSI）

- **適應症**：未禁食、高誤吸風險（急診、飽腹、懷孕、腸阻塞、GERD）
- 步驟：
    1. 預先給氧（preoxygenation）3–5 分鐘 100% O₂
    2. Cricoid pressure（**Sellick maneuver**）：按壓環狀軟骨壓迫食道 → 防止胃內容物逆流
    3. 快速推注誘導藥物 + **Succinylcholine**（最快速起效的肌肉鬆弛劑，onset 30–60 sec）或 Rocuronium 高劑量（1.2 mg/kg）
    4. **不做 bag-mask ventilation**（避免胃脹氣增加嘔吐風險）
    5. 待肌肉鬆弛後迅速插管
    6. 確認管位（ETCO₂）後放開 cricoid pressure

### 2.5 困難呼吸道（Difficult Airway Algorithm）

- **預期困難插管**：清醒插管（awake fiberoptic intubation）→ **gold standard for anticipated difficult airway**
- **非預期困難插管（Cannot Intubate）**：
    - 嘗試不同 blade / video laryngoscopy / bougie
    - 置放 **Supraglottic Airway（SGA/LMA, 喉罩）**
- **Cannot Intubate, Cannot Oxygenate（CICO）**→ **緊急外科呼吸道**：
    - **Cricothyroidotomy（環甲膜切開術）**：穿過 cricothyroid membrane → 最快速的外科呼吸道
    - 位置：甲狀軟骨與環狀軟骨之間
    - **小兒 < 8–12 歲**：建議 needle cricothyroidotomy（環甲膜較小）
    - Tracheostomy：非緊急時使用，需更多時間

### 2.6 喉罩（Laryngeal Mask Airway, LMA）

- 置於下咽部，覆蓋喉頭入口上方
- 優點：不需要喉頭鏡、置入快速、刺激較少
- 缺點：**不能完全防止誤吸**、不適合高呼吸道壓力手術
- 適應症：短時間手術、困難插管的過渡措施、rescue device
- 禁忌：高誤吸風險（相對）、需要高呼吸道壓力

### 2.7 拔管（Extubation）

- 條件：意識恢復、保護性反射回復、肌肉力量恢復（TOF ratio ≥ 0.9）、自主呼吸穩定、氧合良好
- 拔管後喉痙攣（laryngospasm）：
    - 最常見於小兒
    - 處理：100% O₂ + jaw thrust + 正壓呼吸 → 無效時 → Succinylcholine 小劑量（0.1–0.5 mg/kg IV）

## 三、全身麻醉

### 3.1 全身麻醉四要素

1. **意識喪失（Hypnosis/Unconsciousness）**：吸入性或靜脈麻醉劑
2. **鎮痛（Analgesia）**：Opioids
3. **肌肉鬆弛（Muscle relaxation）**：Neuromuscular blocking agents
4. **自主神經反射抑制（Areflexia）**：抑制手術刺激引起的心血管反應

### 3.2 全身麻醉階段

- **誘導（Induction）**：使病人從清醒進入麻醉狀態（通常靜脈給藥最快）
- **維持（Maintenance）**：吸入性麻醉劑 ± 靜脈藥物
- **甦醒（Emergence）**：停止麻醉藥 → 自主恢復

### 3.3 吸入性麻醉劑（Inhalational Anesthetics）

#### MAC（Minimum Alveolar Concentration）

- 定義：在 1 大氣壓下，**50%** 的病人對手術切皮不產生運動反應的肺泡濃度
- MAC 越低 → 效力越強
- MAC 具加成性（多種藥物合併時相加）
- 1.3 MAC → 95% 病人無反應
- 0.3–0.5 MAC → 意識喪失（MAC awake）

#### 影響 MAC 的因素

- **MAC 升高**（需要更多麻醉劑）：年輕（6 個月 MAC 最高）、紅髮、高體溫、慢性酒精使用、安非他命（急性）
- **MAC 降低**（需要更少麻醉劑）：老年、低體溫、低血壓、急性酒精中毒、懷孕、Opioids、Benzodiazepines、α₂ agonists（dexmedetomidine）、低血鈉、高鎂
- **不影響 MAC**：性別、麻醉持續時間、甲狀腺功能

#### 常用吸入性麻醉劑比較

| 藥物 | MAC（%） | 血氣分配係數 | 特點 |
|------|---------|-----------|------|
| **N₂O（笑氣）** | 104 | 0.47（最低） | 效力弱（MAC>100%，單獨無法全麻）、誘導/甦醒最快、second gas effect、擴張封閉腔（氣胸禁用！）、不抑制心血管、可致 megaloblastic anemia（抑制 Vit B₁₂/methionine synthase） |
| **Sevoflurane** | 2.0 | 0.65 | 誘導/甦醒快、**氣味溫和不刺激 → 小兒吸入誘導首選**、可產生 Compound A（腎毒性，需 fresh gas flow ≥ 2 L/min）、可觸發 MH |
| **Desflurane** | 6.0 | 0.42 | 誘導/甦醒最快（血氣分配係數最低的鹵化麻醉劑）、**刺激性強 → 不適合吸入誘導**（咳嗽、喉痙攣、交感興奮）、肥胖/短手術甦醒最快、可觸發 MH |
| **Isoflurane** | 1.15 | 1.46 | 冠狀動脈竊流（coronary steal，理論上）、較強心血管抑制、可觸發 MH |
| Halothane | 0.75 | 2.54 | 誘導/甦醒慢、**肝毒性（halothane hepatitis，免疫介導）**、心肌敏化 catecholamine → 心律不整、可觸發 MH、已少用 |

- **血氣分配係數（Blood:Gas Partition Coefficient）**：越低 → 血液溶解度越低 → 肺泡濃度上升越快 → **誘導與甦醒越快**
- **Oil:Gas Partition Coefficient**：越高 → 脂溶性越高 → **效力越強（MAC 越低）**

#### 吸入性麻醉劑的共同特性

- 所有鹵化吸入麻醉劑（除 N₂O）：
    - 劑量依賴性地降低血壓（心肌抑制 ± 血管擴張）
    - 呼吸抑制（降低 CO₂ 驅動力）
    - 支氣管擴張（尤其 sevoflurane → 氣喘病人有利）
    - 降低腦代謝率但增加腦血流（CBF↑ → ICP↑）→ 神經外科需注意
    - 肌肉鬆弛
    - 降低子宮張力 → 大量使用時增加產後出血風險
    - 可觸發 **Malignant Hyperthermia（MH）**
- **N₂O 的特殊風險**：
    - 擴散至封閉腔隙 → **氣胸、腸阻塞、中耳手術、氣腦** 時禁用
    - Diffusion hypoxia：停用 N₂O 時大量 N₂O 湧入肺泡稀釋 O₂ → 停用時給 100% O₂ 數分鐘

### 3.4 靜脈麻醉劑（Intravenous Anesthetics）

| 藥物 | 作用機制 | Onset | 特點 |
|------|---------|-------|------|
| **Propofol** | GABA_A agonist | 15–30 sec | **最常用誘導劑**；快速、平順誘導/甦醒；降低 ICP/CMRO₂；**止吐效果**；注射痛（lidocaine 可減輕）；低血壓（最常見副作用，血管擴張+心肌抑制）；**無鎮痛效果**；Propofol infusion syndrome（長期高劑量 → 代謝性酸中毒、rhabdomyolysis、高血鉀、心衰竭 → 可致死）；大豆/蛋黃過敏者注意 |
| **Etomidate** | GABA_A agonist | 15–30 sec | **血行最穩定的誘導劑 → 血行不穩/心臟病人首選**；降低 ICP；**抑制腎上腺皮質（11β-hydroxylase）→ 不可長期輸注**；myoclonus；注射痛；無鎮痛 |
| **Ketamine** | NMDA antagonist | 30–60 sec IV | **唯一升高血壓/心率的麻醉劑**（交感興奮）；解離式麻醉；支氣管擴張（氣喘有利）；**有鎮痛效果**；保留自主呼吸及呼吸道保護反射（相對）；升高 ICP/IOP → 顱內壓升高/開放性眼球傷禁忌；分泌物增加；emergence delirium（甦醒譫妄）→ benzodiazepine 可預防；IM 給予亦有效 |
| **Midazolam** | GABA_A agonist (BZD) | 1–3 min | 術前焦慮緩解、sedation；順行性遺忘（anterograde amnesia）；無鎮痛；呼吸抑制（與 opioid 合用加重）；拮抗劑：**Flumazenil** |
| **Thiopental** | GABA_A agonist (barbiturate) | 15–30 sec | 快速誘導；降低 ICP/CMRO₂（腦保護，**status epilepticus 可用**）；心肌抑制/低血壓；**Porphyria 禁用**（acute intermittent porphyria → 觸發急性發作）；已少用 |
| **Dexmedetomidine** | α₂ agonist（central） | 5–10 min | 鎮靜但病人可被喚醒（cooperative sedation）；有鎮痛效果；**不抑制呼吸**（最大特色）；心跳過緩+低血壓；ICU 鎮靜、困難呼吸道 awake intubation 輔助 |

#### TIVA（Total Intravenous Anesthesia）

- 完全使用靜脈藥物維持麻醉，不用吸入性麻醉劑
- 常用組合：**Propofol + Remifentanil**
- 適應症：MH 高風險者、需要避免 N₂O、術後噁心嘔吐高風險、神經外科（ICP 控制）

### 3.5 Opioids（鴉片類鎮痛劑）

| 藥物 | 效力（相對 morphine） | 特點 |
|------|---------------------|------|
| **Morphine** | 1× | 組織胺釋放 → 低血壓/支氣管收縮/搔癢；活性代謝物（M6G）→ 腎功能不全蓄積 |
| **Fentanyl** | 100× | 快速起效、短效；**術中最常用**；胸壁僵硬（chest wall rigidity，大劑量快速注射） |
| **Remifentanil** | 200× | 由血漿酯酶代謝（不經肝腎）→ **超短效**；適合 TIVA；停藥後快速消退 → 需銜接其他止痛 |
| **Sufentanil** | 500–1000× | 效力最強之常用 opioid |
| **Alfentanil** | 10–25× | 起效極快、作用時間短 |
| **Meperidine** | 0.1× | 代謝物 normeperidine → 癲癇（腎功能不全蓄積）；與 **MAOi 嚴重交互作用**（serotonin syndrome / hypertensive crisis → 禁用！）；輕度抑制 serotonin/NE reuptake；止顫作用（抗 shivering） |

#### Opioid 共同效應

- 呼吸抑制（**最危險的副作用**，降低 CO₂ 敏感度）
- 瞳孔縮小（miosis / pinpoint pupils）
- 噁心嘔吐（CTZ 刺激）
- 便秘/尿滯留
- 搔癢（尤其 neuraxial 給予時）
- 膽道括約肌（Sphincter of Oddi）痙攣
- 肌肉僵硬（chest wall rigidity，大劑量 fentanyl）

#### Opioid 拮抗劑

- **Naloxone（Narcan）**：完全拮抗劑（IV 0.04–0.4 mg），半衰期較短 → 可能需要重複給予或持續輸注，因 opioid 可能 outlast naloxone

### 3.6 神經肌肉阻斷劑（Neuromuscular Blocking Agents, NMBAs）

#### 去極化型（Depolarizing）

| 藥物 | 特點 |
|------|------|
| **Succinylcholine（SCh）** | 唯一臨床使用之去極化 NMBA；**onset 最快（30–60 sec）、duration 最短（5–10 min）**；RSI 首選；作用於 nicotinic receptor → 先引起 fasciculation 再癱瘓；由 **pseudocholinesterase（血漿膽鹼酯酶）** 代謝 |

**Succinylcholine 的副作用與禁忌（非常重要！）**：

- **高血鉀**：正常上升 0.5 mEq/L，但在以下情況可致命性高血鉀 → **禁用**：
    - 燒傷（受傷後 24 hr 至癒合數月）
    - 脊髓損傷 / 上運動神經元病變 / 去神經化
    - 長期臥床/不動（>72 小時後開始危險，持續至少 6 個月）
    - 肌肉病變（muscular dystrophy）
    - 嚴重感染（如 tetanus、botulism 恢復期）
    - 機制：上述情況 → 乙醯膽鹼受體上調（upregulation/extrajunctional receptors）→ 鉀大量外流
- **Malignant Hyperthermia（MH）**觸發劑
- **心跳過緩**（尤其小兒、重複給藥 → vagal stimulation）→ 小兒先給 atropine
- **IOP 升高**（開放性眼球傷相對禁忌）
- **ICP 短暫升高**
- **IGP 升高（胃內壓）**：理論上增加誤吸風險（但臨床上 RSI 仍使用）
- **Fasciculation → 肌肉酸痛**
- **Pseudocholinesterase deficiency（血漿膽鹼酯酶缺乏症）**：遺傳性 → SCh 代謝極度延長 → 長時間肌肉癱瘓（數小時）→ 處理：持續呼吸器支持直到自行恢復

#### 非去極化型（Non-depolarizing）

| 藥物 | Onset | Duration | 特點 |
|------|-------|----------|------|
| **Rocuronium** | 1–2 min（高劑量 45–60 sec） | 中效（30–60 min） | 最常用；高劑量可替代 SCh 用於 RSI（1.2 mg/kg）；可被 **Sugammadex** 特異性拮抗 |
| Vecuronium | 2–3 min | 中效 | 無心血管副作用；肝臟代謝 |
| Atracurium | 2–3 min | 中效 | **Hofmann elimination**（自發降解，不經肝腎）→ 肝腎衰竭可安全使用；代謝物 laudanosine（高劑量可致癲癇）；組織胺釋放 |
| Cisatracurium | 2–3 min | 中效 | Atracurium 的 cis 異構物；Hofmann elimination；無組織胺釋放 |
| Pancuronium | 3–5 min | 長效（60–90 min） | Vagolytic → tachycardia；腎臟排泄 |

#### 神經肌肉監測（TOF, Train-of-Four）

- 給予 4 個連續電刺激 → 計算第 4 與第 1 次 twitch 的比值（T4/T1）
- 非去極化阻斷：漸進性遞減（fade），T4/T1 < 1
- 去極化阻斷（Phase I, SCh）：均勻降低，無 fade
- 安全拔管標準：**TOF ratio ≥ 0.9**（T4/T1 ≥ 0.9）
- 殘餘肌肉阻斷（residual curarization）→ 術後呼吸衰竭的重要原因

#### NMBAs 拮抗劑

| 拮抗劑 | 機制 | 適用 |
|--------|------|------|
| **Neostigmine** | Anticholinesterase（增加 NMJ 的 ACh 濃度）| 所有非去極化 NMBA；**必須合併 glycopyrrolate 或 atropine**（防止 muscarinic side effects：心跳過緩、唾液增加、支氣管收縮） |
| **Sugammadex** | 直接包裹 rocuronium/vecuronium（modified γ-cyclodextrin）| **Rocuronium/Vecuronium 專用**；起效快（1–3 min）；可逆轉深度阻斷；不需合併 atropine；可用於 RSI 劑量 rocuronium 的快速逆轉 → 改變困難呼吸道管理策略 |

#### Myasthenia Gravis（MG）與 NMBA

- MG 病人對 **非去極化 NMBA 極度敏感**（NMJ 的 AChR 數量減少）→ 劑量須大幅減少或避免使用
- 對 **Succinylcholine 則有抗性**（receptor 少 → 需更高劑量才能去極化）
- 拮抗首選：**Sugammadex**（避免 neostigmine 的不可預測性）

## 四、區域麻醉（Regional Anesthesia）

### 4.1 局部麻醉劑（Local Anesthetics）

#### 分類

| 類型 | 藥物 | 記法 |
|------|------|------|
| **Ester 類** | Procaine、Chloroprocaine、Tetracaine、Cocaine | 名稱有一個 "i"（或無 "i"）→ 記法：ester = 1 個 "i" |
| **Amide 類** | **L**idocaine、**B**upivacaine、Ropivacaine、Mepivacaine、Levobupivacaine | 名稱有兩個 "i" → 記法：am**i**de = 2 個 "i" |

- Ester 類由 **pseudocholinesterase** 在血漿中代謝 → 代謝產物 **PABA** → 過敏反應較常見
- Amide 類由 **肝臟** 代謝 → 過敏極罕見

#### 作用機制

- 阻斷電壓門控鈉離子通道（voltage-gated Na⁺ channel）→ 抑制神經衝動傳導
- **阻斷順序（sensitivity）**：自主神經（B fiber, sympathetic）→ 痛/溫覺（C fiber, Aδ）→ 觸壓覺（Aβ）→ 運動神經（Aα）
    - 意義：感覺阻斷先於運動阻斷；交感先被阻斷 → **低血壓**
- **恢復順序**：反過來，運動先恢復、自主神經最後恢復

#### 常用藥物特性

| 藥物 | Onset | Duration | 特點 |
|------|-------|----------|------|
| **Lidocaine** | 快 | 1–2 hr | 最常用；可 IV 抗心律不整；加 epinephrine 可延長至 2–4 hr |
| **Bupivacaine** | 慢 | 3–8 hr | 長效；**心臟毒性最強**（cardiac arrest 難以復甦）；sensory > motor block |
| **Ropivacaine** | 中 | 3–8 hr | 長效；心臟毒性較 bupivacaine 低；較少運動阻斷 |
| **Levobupivacaine** | 中 | 3–8 hr | Bupivacaine 的 S-enantiomer；心臟毒性較低 |
| **Chloroprocaine** | 最快 | 30–60 min | 最短效；安全性高（快速水解） |

#### Epinephrine 添加

- 作用：局部血管收縮 → 延緩吸收 → **延長作用時間 + 減少全身毒性 + 減少出血**
- 濃度：1:200,000（5 μg/mL）
- **禁忌**：末端動脈區域（手指、腳趾、鼻尖、耳垂、陰莖 → "fingers, toes, nose, ears, hose"）→ 可致缺血壞死（但近年文獻對 digital block 加 epinephrine 的安全性有爭議）
- 局部麻醉劑的最大劑量（含 vs 不含 epinephrine）：
    - Lidocaine：4.5 mg/kg（不含 epi）→ 7 mg/kg（含 epi）
    - Bupivacaine：2.5 mg/kg（不含 epi）→ 3 mg/kg（含 epi）

#### 局部麻醉劑全身毒性（LAST, Local Anesthetic Systemic Toxicity）

- 原因：意外血管內注射或劑量過大 → 血中濃度過高
- **CNS 症狀先於心臟症狀**（CNS 對局麻劑更敏感）：
    - CNS（興奮→抑制）：口周麻木、耳鳴、金屬味、眩暈 → **癲癇** → 意識喪失 → 呼吸停止
    - CVS：心跳過緩、低血壓、傳導阻斷 → **心律不整 → 心跳停止**
    - **Bupivacaine** 的心臟毒性最強且最難復甦（R-enantiomer 與 Na⁺ channel 結合強）
- **治療**：
    - 停止注射、呼叫幫助
    - ABC：給氧、必要時插管
    - 癲癇：Benzodiazepine（midazolam）→ 避免用 propofol 大劑量（加重心肌抑制）
    - **20% Intralipid（脂肪乳劑）**→ 作為 "lipid sink" 吸附脂溶性局麻劑 → **LAST 的特效解毒劑**
        - Bolus 1.5 mL/kg → 接續 0.25 mL/kg/min 持續輸注
    - CPR（如果心跳停止）：避免用 vasopressin、避免大劑量 epinephrine；考慮 cardiopulmonary bypass/ECMO

### 4.2 脊髓麻醉（Spinal Anesthesia = Subarachnoid Block, SAB）

- 將局部麻醉劑注入 **蛛網膜下腔（subarachnoid space）**，直接作用於脊髓神經根
- 穿刺位置：**L3–L4 或 L4–L5 間隙**（成人脊髓終止於 L1–L2 → 穿刺在此以下避免傷及脊髓）
    - 定位：兩側髂脊最高點連線（Tuffier's line）通過 L4 棘突或 L4–L5 間隙
- 穿過層次（由外而內）：皮膚 → 皮下 → 棘上韌帶 → 棘間韌帶 → **黃韌帶（ligamentum flavum）** → **硬膜外腔** → **硬腦膜（dura mater）** → **蛛網膜（arachnoid）** → 蛛網膜下腔（CSF 流出確認位置）
- 常用藥物：**Bupivacaine（heavy/hyperbaric）**最常用
- 作用：起效快（5 分鐘內）、阻斷完全、持續 1.5–3 小時
- 阻斷高度受影響因素：
    - **藥液的比重（baricity）**：hyperbaric → 受重力影響往低處流，體位影響阻斷範圍
    - 藥物劑量（最重要）
    - 病人身高、注射速度

#### 適應症

- 下腹部、骨盆、會陰部、下肢手術
- 剖腹產（常選 spinal 或 CSE）

#### 絕對禁忌症

- 病人拒絕
- 穿刺部位感染
- **凝血障礙**（INR↑、血小板過低、使用抗凝血劑未按時停藥）
- 嚴重低血容（hypovolemia）→ 交感阻斷 → 致命性低血壓
- 顱內壓升高（有 herniation 風險）

#### 併發症

- **低血壓**：**最常見的併發症**（交感神經阻斷 → 血管擴張 → venous return↓）
    - 預防/治療：輸液、**Ephedrine**（混合 α+β agonist）或 **Phenylephrine**（α₁ agonist，產科首選因不影響胎兒血流）
- **Post-dural puncture headache（PDPH，硬膜穿刺後頭痛）**：
    - 機制：硬膜穿孔 → CSF 漏出 → 顱內壓降低 → 腦膜/血管牽拉
    - 特徵：**姿勢性頭痛**（坐/站時加重、平躺緩解）+ 頸部僵硬、噁心
    - 風險因素：年輕女性、大號針頭、cutting needle（vs pencil-point/atraumatic needle）
    - 治療：保守（臥床、水分、caffeine、止痛藥）→ 無效 → **Epidural blood patch**（最有效治療，用自體血 15–20 mL 注入硬膜外腔封堵穿孔）
- **高位脊髓麻醉（Total spinal）**：阻斷過高 → 呼吸肌癱瘓 + 嚴重低血壓 → 需緊急插管及升壓
- 尿滯留（S2–S4 副交感阻斷 → 最晚恢復的功能）
- 暫時性神經症狀（TNS）：尤其使用 lidocaine spinal → 臀部/下肢放射痛
- 硬膜外血腫/膿瘍（罕見但嚴重 → 脊髓壓迫 → 需緊急手術減壓）

### 4.3 硬膜外麻醉（Epidural Anesthesia）

- 將藥物注入 **硬膜外腔（epidural space）**，作用於穿出硬膜的脊髓神經根
- 可在各節段施行（cervical 很少用、thoracic、lumbar、sacral/caudal）
- 與 spinal 的關鍵差異：
    - **不穿破硬膜**（stop at epidural space）
    - **可置入 catheter → 持續給藥**（可用於術後止痛、產科鎮痛）
    - 起效較慢（15–20 分鐘）
    - 需要較大劑量藥物
    - 阻斷可分節段調整（segmental block）

#### 硬膜外腔確認法

- **Loss of resistance（LOR）**：推進 Tuohy 針同時注入生理食鹽水或空氣 → 進入硬膜外腔時阻力突然消失 → **最常用的確認方法**
- Hanging drop method：黃韌帶至硬膜外腔的負壓吸入懸滴

#### 適應症

- 術後止痛（thoracic epidural → 胸腹部大手術止痛 gold standard）
- **產科無痛分娩**：T10–L1（子宮收縮痛）→ 第二產程加上 S2–S4（會陰痛）
- 手術麻醉（可單獨或合併全麻）

#### 併發症

- 低血壓（同 spinal，但較緩和）
- 意外硬膜穿破 → PDPH（Tuohy 針較粗 → PDPH 更嚴重）
- 意外蛛網膜下注射 → total spinal
- 意外血管內注射 → LAST
- 硬膜外血腫/膿瘍
- 局麻劑經胎盤 → 新生兒抑制（大劑量時）

### 4.4 Combined Spinal-Epidural（CSE）

- 同一次穿刺先做 spinal（快速起效）→ 再放 epidural catheter（持續給藥延長）
- 常用於剖腹產、長時間下肢手術

### 4.5 Caudal Block（尾側阻滯）

- 經 sacral hiatus 注入硬膜外腔
- **小兒常用**：用於臍部以下手術的術後止痛（疝氣修補、陰囊手術、下肢手術）
- 解剖標記：sacral cornua（兩個骨性突起）之間

### 4.6 神經阻斷（Peripheral Nerve Blocks）

- 在超音波導引下注射局麻劑到特定神經周圍
- 常見阻滯：
    - **Brachial plexus block**（上肢手術）：
        - Interscalene：肩部/上臂手術（C5–C7 為主，可能 miss C8–T1 → ulnar nerve）；**膈神經阻斷（phrenic nerve block）100%** → 對側肺功能不全者禁忌
        - Supraclavicular：上肢全覆蓋；氣胸風險
        - Infraclavicular：肘部以下
        - Axillary：前臂/手部；最安全（無氣胸/膈神經風險）
    - **Femoral nerve block**：大腿前側、股四頭肌手術、髖關節/膝關節鎮痛
    - **Sciatic nerve block**：大腿後側、小腿、足部
    - **TAP block（Transversus Abdominis Plane）**：腹壁手術止痛
    - **Fascia iliaca block**：髖關節骨折止痛

## 五、產科麻醉

### 5.1 懷孕的生理變化（麻醉相關）

| 系統 | 變化 | 臨床意義 |
|------|------|----------|
| 心血管 | CO↑ 30–50%、HR↑、SVR↓、BP 輕微↓ | 可耐受較多失血但一旦代償不了急遽惡化 |
| 呼吸 | MV↑（TV↑為主）、FRC↓、O₂ 消耗↑ | **缺氧耐受力下降 → 去氧飽和更快 → 預氧合更重要**；過度換氣 → 輕度 respiratory alkalosis（PaCO₂ ~30 mmHg 正常） |
| 消化 | 胃排空延遲（progesterone）、LES 張力↓、胃酸分泌↑ | **所有孕婦視為 full stomach → RSI 適應症** |
| 血液 | 血量↑ 40–50%（血漿>紅血球）→ 稀釋性貧血、生理性高凝狀態 | Hb 可低至 10–11 g/dL 正常；DVT/PE 風險↑ |
| 神經 | 局麻劑敏感度↑、硬膜外靜脈叢充血 → epidural space↓ | 需要較少劑量的局麻劑 |
| 子宮 | Aortocaval compression（仰臥低血壓症候群） | **>20 週孕婦仰臥 → 子宮壓迫 IVC/aorta → CO↓、低血壓** → **左側傾斜 15°（left uterine displacement）** |

### 5.2 產科鎮痛與麻醉

#### 自然產無痛分娩

- **Epidural analgesia**：最常用、最有效的分娩止痛
- 時機：產婦要求時即可（不需等子宮頸開指特定大小）
- 藥物：低濃度 bupivacaine/ropivacaine + fentanyl/sufentanil
- 不影響剖腹產率（研究證實）

#### 剖腹產麻醉

- **首選：區域麻醉（Spinal or CSE）**
    - Spinal：bupivacaine（heavy）+ fentanyl/morphine → 快速起效
    - 阻斷高度需至 **T4**（乳頭平面）以提供足夠的手術範圍
- 全身麻醉適應症：產婦拒絕區域麻醉、凝血異常、嚴重低血容、胎兒緊急危難（無時間做 spinal）、區域麻醉失敗
    - 全身麻醉風險：困難插管（呼吸道水腫 → **使用較小口徑 ETT，通常 6.0–6.5 mm**）、aspiration、子宮鬆弛（吸入麻醉劑 → 產後出血↑）、新生兒呼吸抑制

## 六、惡性高熱（Malignant Hyperthermia, MH）

### 6.1 病因與機制

- **遺傳**：**體染色體顯性（AD）**，RYR1 基因突變（ryanodine receptor）
- 機制：觸發劑 → 骨骼肌 RyR1 異常開啟 → 肌漿網 Ca²⁺ 大量釋出 → 肌肉持續收縮 + 代謝亢進
- **觸發劑**：
    - 所有 **揮發性吸入麻醉劑**（halothane、isoflurane、sevoflurane、desflurane）
    - **Succinylcholine**
- **不觸發 MH 的安全藥物**：N₂O、所有靜脈麻醉劑（propofol、etomidate、ketamine、barbiturates）、opioids、benzodiazepines、非去極化 NMBAs、局部麻醉劑

### 6.2 臨床表現

- **最早的徵象**：**ETCO₂ 急遽升高**（CO₂ 產量大增）+ 不明原因心搏過速
- **Masseter muscle rigidity（嚼肌僵直）**：尤其在 SCh 給予後 → 為 MH 早期警訊
- 高燒（可快速升至 >40°C，每 15 分鐘升 1–2°C）
- 肌肉僵硬（generalized rigidity）
- 混合型酸中毒（respiratory + metabolic acidosis）
- Rhabdomyolysis → myoglobinuria → 暗色尿
- **高血鉀**
- 心律不整（tachycardia → VT → VF）
- DIC（晚期）

### 6.3 治療

1. **立即停用所有觸發劑**（關閉揮發性麻醉劑，更換新鮮管路和 CO₂ 吸收罐）
2. **Dantrolene**（唯一特效藥）：2.5 mg/kg IV bolus → 每 5 分鐘重複至症狀控制（最高可用 10 mg/kg）
    - 機制：直接作用於 RyR1 → 抑制 Ca²⁺ 從肌漿網釋出
3. **100% O₂，高流量通氣**（排出 CO₂）
4. **積極降溫**：冰毯、冰鹽水灌注（IV、NG、bladder、peritoneal）
5. 矯正 **高血鉀**：insulin + glucose、NaHCO₃、calcium gluconate（避免 calcium channel blockers → 與 dantrolene 交互作用可致心跳停止）
6. 監測 CK、myoglobin、電解質、ABG、尿量
7. 大量輸液維持尿量 > 2 mL/kg/hr（預防 myoglobin 腎損傷）
8. 術後入 ICU 觀察至少 24–72 小時（可能復發）

### 6.4 診斷確認

- **Caffeine-halothane contracture test（CHCT）**：取肌肉活檢 → 暴露於 caffeine 和 halothane → 陽性反應即確診 → **gold standard**
- 基因檢測（RYR1）：可用但不完全排除（可有 false negative）

### 6.5 MH 易感病人的麻醉

- **TIVA**：propofol + remifentanil + 非去極化 NMBA
- 使用無揮發性麻醉劑的清潔麻醉機（flush > 10–20 分鐘或使用備用乾淨的管路）
- 準備 dantrolene 在旁

## 七、術中監測

### 7.1 ASA 標準監測（Standard Monitors）

- **氧合**：Pulse oximetry（SpO₂）、FiO₂ analyzer
- **通氣**：ETCO₂（capnography）、呼吸音聽診、胸廓起伏
- **循環**：ECG（持續）、BP（至少每 5 分鐘）、心音
- **體溫**：長時間手術或小兒/老人
- 每次麻醉中至少有一位有資格的麻醉人員持續在場

### 7.2 Pulse Oximetry（脈搏血氧儀）

- 原理：利用 oxyhemoglobin 與 deoxyhemoglobin 對兩種光波長（紅光 660nm、紅外光 940nm）的吸收比例不同來計算 SpO₂
- **不準確的情況**：
    - **CO 中毒（COHb）**：pulse ox **假性偏高**（COHb 被讀成 OxyHb）
    - **MetHb（methemoglobinemia）**：SpO₂ 趨向 **85%**（不論實際 SaO₂）
    - 低灌流狀態（休克、低體溫、blood pressure cuff 壓住）
    - 指甲油（深色，尤其藍/黑色）
    - 嚴重貧血（極低 Hb 時）
    - 強光干擾、electrical noise

### 7.3 Capnography（ETCO₂ 監測）

- **最可靠的確認氣管內管位置的方法**
- 正常 ETCO₂：35–45 mmHg
- **ETCO₂ 升高**：MH、CO₂ 吸收不良、通氣不足、NaHCO₃ 給予、增加 CO₂ 產量
- **ETCO₂ 降低**：過度通氣、低心輸出量、肺栓塞、管路脫接、心跳停止
- **ETCO₂ 突然降至零**：管路脫接、食道插管、管路阻塞、呼吸器故障
- CPR 時 ETCO₂ 可監測胸外按壓品質（ETCO₂ <10 mmHg → 灌流不足）
- ETCO₂ 突然升高 → 考慮 ROSC（恢復自發循環）

### 7.4 動脈導管（Arterial Line, A-line）

- 適應症：血行不穩病人、需頻繁 ABG、大手術、控制性低血壓
- 最常用位置：**橈動脈（radial artery）**
    - 穿刺前做 **Allen's test**：同時壓迫橈動脈和尺動脈 → 放開尺動脈 → 手掌應在 5–7 秒內恢復紅潤 → 表示 ulnar collateral 足夠 → 安全穿刺橈動脈
- 提供：beat-to-beat BP 監測、動脈波形分析、ABG 取樣

### 7.5 中心靜脈導管（Central Venous Catheter, CVC）

- 適應症：大量輸液/升壓劑給予、CVP 監測、TPN、周邊 IV 困難
- 常用位置：internal jugular vein（IJV, 最常用）、subclavian vein、femoral vein
- 併發症：氣胸（subclavian 最高風險）、動脈穿刺、空氣栓塞、感染、心律不整（導管觸及心臟）

### 7.6 BIS（Bispectral Index）監測

- 用腦電圖（EEG）衍生的數值監測麻醉深度
- 範圍 0–100：
    - 100：完全清醒
    - 40–60：**全身麻醉適當深度**
    - <40：過深（burst suppression → 平坦腦波）
    - 0：無腦電活動
- 目的：減少術中知曉（awareness）、避免過深麻醉

### 7.7 體溫監測

- 麻醉中低體溫常見（全麻 → 體溫調節抑制 + 血管擴張；**局麻/區域麻醉 → 交感阻斷 → 血管擴張**；手術室低溫；冷輸液/腹腔暴露）
- 低體溫的危害：凝血障礙（最重要）、感染率增加、代謝下降、藥物代謝延長、心律不整、寒顫（shivering → O₂ 消耗增加 200–400%）
- **最核心體溫監測點**：肺動脈（最準確，但侵入性）、食道下段、鼻咽、鼓膜
- 預防：warm blanket（forced-air warming 最有效）、加溫輸液、提高室溫

## 八、術後疼痛處理（Postoperative Pain Management）

### 8.1 疼痛評估

- VAS（Visual Analog Scale）：0–10
- NRS（Numeric Rating Scale）
- Wong-Baker Faces：小兒/語言障礙
- 評估頻率：視為第五生命徵象

### 8.2 WHO 疼痛階梯（三階段）

| 階段 | 疼痛程度 | 藥物 |
|------|---------|------|
| 第一階 | 輕度 | 非鴉片類（Acetaminophen、NSAIDs） |
| 第二階 | 中度 | 弱鴉片類（Tramadol、Codeine）± 非鴉片類 |
| 第三階 | 重度 | 強鴉片類（Morphine、Fentanyl）± 非鴉片類 |

### 8.3 多模式止痛（Multimodal Analgesia）

- 使用不同機制的止痛藥組合 → 減少個別藥物劑量 → 降低副作用
- **Acetaminophen**：中樞 COX 抑制；肝毒性（最大劑量 4 g/day）
- **NSAIDs（Ketorolac, Ibuprofen, Diclofenac）**：抑制 COX → 減少 prostaglandin → 抗炎/鎮痛
    - 副作用：腎損傷、胃潰瘍/出血、血小板功能抑制、影響骨癒合（相對）
    - 禁忌：腎功能不全、消化性潰瘍、凝血異常、aspirin-exacerbated respiratory disease
- **Gabapentin/Pregabalin**：α₂δ calcium channel 調節；神經病變痛、術後輔助止痛
- **Ketamine**（低劑量 sub-anesthetic）：NMDA antagonist；opioid tolerance 者有效、預防慢性術後痛
- **Dexamethasone**：單劑可減少術後噁心嘔吐和術後止痛需求

### 8.4 Patient-Controlled Analgesia（PCA）

- 病人自行按壓給藥（通常 IV morphine 或 fentanyl）
- 設定：demand dose + lockout interval（通常 5–10 min）± 持續輸注（basal rate）
- 安全性：lockout 機制防止過量；清醒度降低時不會按壓
- **不可由家屬或護理人員代按！**（PCA by proxy → 呼吸抑制風險↑）

### 8.5 Epidural Analgesia（硬膜外止痛）

- 胸腹部大手術的術後止痛 **gold standard**
- 藥物：低濃度 local anesthetic（bupivacaine 0.0625–0.125%）+ opioid（fentanyl/morphine）
- 優點：優異止痛、減少全身 opioid、促進術後腸道恢復、改善肺功能（減少 atelectasis）
- 副作用：低血壓、尿滯留、搔癢（opioid）、運動阻斷（local anesthetic 濃度太高）、呼吸抑制（neuraxial opioid → 延遲型，尤其 morphine，需密切監測）

## 九、術後噁心嘔吐（PONV）

### 9.1 風險因子

- **病人因素**：女性（最強風險因子）、非吸菸者、PONV/暈車史、年輕
- **麻醉因素**：吸入性麻醉劑（N₂O 尤甚）、opioids（術中+術後）
- **手術因素**：腹腔鏡手術、婦科手術、斜視手術（小兒）、中耳手術、長時間手術

### 9.2 Apfel Simplified Risk Score

- 四項風險因子（每項 1 分）：❶ 女性、❷ 非吸菸、❸ PONV/暈車史、❹ 術後使用 opioids
- 預估 PONV 發生率：0 分 ≈ 10%、1 分 ≈ 20%、2 分 ≈ 40%、3 分 ≈ 60%、4 分 ≈ 80%

### 9.3 預防與治療

| 藥物 | 機制 | 特點 |
|------|------|------|
| **Ondansetron** | 5-HT₃ antagonist | 最常用止吐藥；QT prolongation |
| **Dexamethasone** | 類固醇 | 誘導時給予；增強其他止吐藥效果；高血糖 |
| **Droperidol** | D₂ antagonist（butyrophenone） | 高效；QT prolongation（黑框警告）；EPS |
| **Scopolamine** | Muscarinic antagonist | 貼片（耳後）；暈動症有效；口乾/視力模糊/老人譫妄 |
| **Metoclopramide** | D₂ antagonist + 促胃動力 | 較弱止吐；EPS（extrapyramidal symptoms） |
| **Aprepitant** | NK₁ receptor antagonist | 高效但貴 |

- **高風險病人（≥2 分）→ 多藥物預防**：不同機制的止吐藥合併使用
- **TIVA（propofol-based）**：本身有止吐效果 → 高風險 PONV 病人考慮
- 減少 N₂O、減少 opioid、充分水分補充

## 十、術中常見危機處理

### 10.1 麻醉中過敏反應（Anaphylaxis）

- 最常見觸發藥物：**NMBAs（尤其 rocuronium、succinylcholine）** > latex > 抗生素
- 表現：低血壓（最常見心血管表現）、心搏過速、支氣管痙攣、皮疹/蕁麻疹（但全麻下可能被遮蓋）
- 治療：
    1. 停止可疑藥物
    2. **Epinephrine**（首選，IV 10–100 μg 逐步給予，IM 0.3–0.5 mg）
    3. 大量輸液（crystalloid 1–2 L bolus）
    4. 支氣管痙攣：albuterol 吸入
    5. 類固醇、抗組織胺（H₁ + H₂ blocker）：次要治療
    6. 送 serum tryptase（確認 mast cell 活化）

### 10.2 支氣管痙攣（Bronchospasm）

- 術中原因：氣管插管刺激（尤其麻醉過淺）、aspiration、過敏、氣喘
- 表現：呼氣延長（expiratory wheeze）、high peak airway pressure、ETCO₂ 波形上升段斜率改變
- 處理：加深麻醉（sevoflurane 有支氣管擴張作用）、albuterol MDI via ETT、epinephrine（嚴重時）

### 10.3 空氣/氣體栓塞（Air Embolism）

- 高風險手術：**坐姿神經外科手術（sitting craniotomy）**、頸部/肩部手術（血管高於心臟）
- 監測：**TEE（經食道心臟超音波）**（**敏感度最高**）、**precordial Doppler**（敏感度第二，非侵入性首選）、ETCO₂ 突然下降
- 處理：
    1. 淹水手術野（flood surgical field）
    2. 通知外科醫師
    3. 停止 N₂O（會擴大氣泡）
    4. 左側頭低位（Durant maneuver）
    5. 嘗試從 CVC 抽出空氣
    6. 支持治療

### 10.4 喉痙攣（Laryngospasm）

- 聲帶非自主性緊閉 → 完全或部分呼吸道阻塞
- 常見於拔管期、小兒（最常見）
- 觸發：分泌物、血液、淺麻醉下的呼吸道刺激
- 處理：
    1. 移除刺激物（抽吸分泌物）
    2. 100% O₂ + jaw thrust + 持續正壓呼吸（CPAP）
    3. **Larson's notch 壓迫**：在下頜角後方、乳突前方的凹陷處用力按壓
    4. 上述無效 → **Succinylcholine 0.1–1 mg/kg IV**（or 4 mg/kg IM）
    5. 必要時重新插管

### 10.5 術中心跳停止

- **麻醉相關心跳停止**：藥物過量、呼吸道喪失、LAST、MH、高位脊麻、anaphylaxis
- 遵循 ACLS 流程
- 特殊考量：排除可逆原因（6H6T）、確認管路/通氣/藥物正確

## 十一、特殊族群麻醉

### 11.1 小兒麻醉

#### 解剖生理特點

- **呼吸道**：頭大、舌大、會厭軟且 omega 形（Ω）、**喉頭位置較高（C3–C4 vs 成人 C4–C5）**、**聲門下區（cricoid ring）為最狹窄處**（成人為聲門）→ uncuffed ETT 傳統上用於 <8 歲（現今 cuffed ETT 亦安全使用於小兒）
- **呼吸**：呼吸速率快、FRC 小、O₂ 消耗高 → **去氧飽和極快**
- **心血管**：心輸出量 rate-dependent（嬰幼兒心臟順應性低 → 無法有效增加 SV）→ **心跳過緩 = 心輸出量下降 = 危急** → **小兒最常見的心跳停止原因是 bradycardia（通常因缺氧）**
    - 處理：給氧通氣 → Atropine 0.02 mg/kg → Epinephrine
- **體溫調節**：表面積/體重比大 → 失溫快 → 需積極保溫
- **藥理**：MAC 在 6 個月時最高 → 之後隨年齡下降

#### 小兒麻醉重點

- 吸入誘導（inhalation induction）：**Sevoflurane** 首選（氣味溫和、不刺激）
- ETT 固定深度（口角）：大約為 ETT size × 3（例如 4.0 mm → 固定於口角 12 cm）
- IV 輸液：4-2-1 rule
- 常見手術的 caudal block 劑量：0.5–1 mL/kg 的 0.25% bupivacaine

### 11.2 老年人麻醉

- 藥物代謝下降 → 需減少劑量
- MAC 隨年齡下降（每 10 歲下降約 6%）
- 心血管代償能力差 → 對低血壓敏感
- 腎/肝功能下降 → 藥物排除延長
- **術後譫妄（Postoperative delirium）**高風險 → 避免 benzodiazepine（長效）、anticholinergic 藥物

### 11.3 肥胖病人麻醉

- **困難呼吸道**風險↑（困難面罩通氣+困難插管）
- FRC↓ → 去氧飽和更快 → 預氧合重要、head-up/ramped position
- **藥物劑量計算**：
    - Succinylcholine：用 **total body weight（TBW）**
    - 非去極化 NMBAs：用 **ideal body weight（IBW）**
    - 大部分靜脈麻醉劑：用 **lean body weight** 或 IBW
- OSA（obstructive sleep apnea）常見 → 術後呼吸監測、減少 opioid
- Desflurane/TIVA → 甦醒較快（脂溶性低/不蓄積）

## 十二、疼痛與鴉片類藥物

### 12.1 疼痛分類與傳導

| 分類 | 特徵 | 代表 |
|------|------|------|
| **體感性（Somatic）** | 定位明確、銳痛/鈍痛 | 骨折、切割傷 |
| **內臟性（Visceral）** | 定位模糊、悶痛/絞痛、常有 referred pain | 膽絞痛、腸絞痛 |
| **神經病變性（Neuropathic）** | 燒灼、電擊、allodynia、hyperalgesia | 糖尿病神經病變、帶狀疱疹後神經痛、幻肢痛 |

- **Allodynia**：正常不痛的刺激引起疼痛；**Hyperalgesia**：對痛刺激反應加劇
- **Referred pain 經典對應**：心臟 → 左肩/左臂（T1–T4）、橫膈 → 肩部（C3–C5）、膽囊 → 右肩胛（T6–T9）、闌尾初期 → 臍周（T10）
- 痛覺纖維：**Aδ fiber**（快痛、銳痛、薄髓鞘）、**C fiber**（慢痛、鈍痛、無髓鞘）→ 脊髓後角 → 對側 **spinothalamic tract** → 丘腦 → 大腦皮質

### 12.2 鴉片類受體與藥物分類

#### 受體

| 受體 | 主要效應 |
|------|---------|
| **μ（mu）** | **鎮痛（最主要）**、呼吸抑制、欣快感、便秘、瞳孔縮小、身體依賴 |
| κ（kappa） | 鎮痛（脊髓）、鎮靜、dysphoria |
| δ（delta） | 鎮痛調節、情緒 |

#### 藥物分類

| 類別 | 藥物 | 要點 |
|------|------|------|
| Pure agonist | Morphine、Fentanyl、Remifentanil、Hydromorphone、Methadone、Meperidine、Codeine | μ 完全激動 |
| Partial agonist | **Buprenorphine** | μ 部分激動 → ceiling effect；與 full agonist 合用可能拮抗；成癮治療用 |
| Mixed agonist-antagonist | Nalbuphine、Butorphanol、Pentazocine | κ agonist + μ antagonist；可 precipitate withdrawal |
| Pure antagonist | **Naloxone**（IV 短效）、**Naltrexone**（PO 長效） | 完全拮抗 |

### 12.3 常用藥物比較

| 藥物 | 相對效力 | 關鍵特點 |
|------|---------|---------|
| **Morphine** | 1× | 組織胺釋放（低血壓/搔癢）；活性代謝物 **M6G** → 腎功能不全蓄積；膽道痙攣 |
| **Fentanyl** | 100× | 術中最常用；無組織胺釋放；高劑量快注 → **chest wall rigidity** → 需 NMBA |
| **Remifentanil** | ~200× | 血漿酯酶代謝（不經肝腎）→ context-sensitive half-time 恆定 ~3 min；TIVA 首選；停藥後需銜接止痛 |
| **Hydromorphone** | 5–7× | 腎功能不全較安全（無 M6G） |
| **Methadone** | 1–3× | 半衰期極長（15–60 hr）；NMDA antagonist 特性；**QT prolongation**；成癮戒斷治療 |
| **Meperidine** | 0.1× | 代謝物 **normeperidine → 癲癇**；**MAOi 絕對禁忌**（serotonin syndrome）；抗 shivering |
| **Codeine** | 0.1× | 須經 **CYP2D6** 轉化為 morphine → poor metabolizer 無效、ultra-rapid metabolizer 中毒 |
| **Tramadol** | — | 弱 μ agonist + SNRI → serotonin syndrome 風險、降低癲癇閾值 |

#### 共同副作用

- **呼吸抑制**（最危險）、**瞳孔縮小**（不產生耐受性）、噁心嘔吐、**便秘**（最常見，不產生耐受性）、尿滯留、搔癢（neuraxial 給予尤甚）、膽道痙攣
- Opioid 戒斷：流淚流鼻水、腹瀉、piloerection、肌肉痛 → **不舒服但不致命**（cf. 酒精/BZD 戒斷可致命）

### 12.4 中毒與拮抗

- **Opioid 中毒三聯徵**：意識下降 + 呼吸抑制 + **針狀瞳孔**
- **Naloxone**：0.04–0.4 mg IV 逐量滴定（一次大劑量 → 急性戒斷 → 肺水腫/心律不整）
    - Duration 30–45 min → **短於大多數 opioids → 可能需重複給予或持續輸注**
