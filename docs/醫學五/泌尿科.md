# 泌尿科

---

## 一、腎臟與泌尿道解剖生理

### 1.1 腎臟解剖

- 位於後腹腔（retroperitoneal），T12–L3
- **右腎較左腎低**（被肝臟壓下）
- 腎臟血管走行（由前到後）：**Renal vein → Renal artery → Renal pelvis**（VAP 口訣，前→後）
- **左腎靜脈較長**，跨過 aorta 前方（在 SMA 與 aorta 之間）
    - 可被壓迫 → **Nutcracker syndrome**（左腎靜脈受壓 → 血尿、左側精索靜脈曲張）
    - 腎移植時偏好取**左腎**（靜脈較長，吻合較容易）
- 腎動脈為 **end artery**（無側枝循環），阻塞即壞死
- 腎臟節段（Brödel's line）：前後血管分水嶺在腎臟後外側 2/3 → 腎切開術（nephrotomy）應沿此無血管平面

### 1.2 輸尿管解剖

- 長約 25–30 cm，retroperitoneal
- **三個生理狹窄處**（結石最易卡住之處）：
    1. **UPJ（ureteropelvic junction）**：腎盂輸尿管交界
    2. **跨越髂血管處**（crossing iliac vessels）
    3. **UVJ（ureterovesical junction）**：輸尿管膀胱交界 → **最狹窄處**
- 血供：上 1/3 ← 腎動脈；中 1/3 ← gonadal artery、aorta；下 1/3 ← 內髂動脈（internal iliac / vesical arteries）
- 手術原則：輸尿管上段的血供從**內側**來，下段從**外側**來 → 分離輸尿管時注意保護相應方向的血供

### 1.3 膀胱解剖

- 位於骨盆腔內，extraperitoneal（頂部被腹膜覆蓋 → intraperitoneal rupture 常在 dome）
- **三角區（trigone）**：兩側輸尿管開口 + 尿道內口 → 黏膜光滑無皺襞
- 膀胱容量：成人約 400–500 mL
- 支配神經：
    - **副交感（S2–S4, pelvic nerve）**：逼尿肌（detrusor）收縮 → **排尿**
    - **交感（T10–L2, hypogastric nerve）**：β₃ receptor → detrusor 放鬆；α₁ receptor → 內括約肌（bladder neck）收縮 → **儲尿**
    - **體神經（S2–S4, pudendal nerve）**：外括約肌（voluntary）→ 自主控制

### 1.4 攝護腺（Prostate）解剖

- 位於膀胱頸下方，包圍尿道
- 重量正常約 20 g
- **McNeal 分區**：
    - **Transition zone（移行帶）**：5–10% → **BPH 好發區**
    - **Central zone（中央帶）**：25%
    - **Peripheral zone（周邊帶）**：70% → **攝護腺癌好發區**（也是 DRE 可摸到的區域）
- 血供：主要來自 inferior vesical artery
- 靜脈：Santorini's plexus（dorsal venous complex）→ 攝護腺根除術（radical prostatectomy）出血主要來源

### 1.5 睪丸與陰囊

- 胚胎學：睪丸由 genital ridge 發育，原位於後腹腔 → 經腹股溝管下降至陰囊
- **睪丸動脈**：直接從 aorta 分出（L2 高度）
- **靜脈回流**：左精索靜脈 → **左腎靜脈**；右精索靜脈 → **IVC**
    - 左側精索靜脈曲張遠比右側常見（因直角注入左腎靜脈，壓力較高）
    - **新發的右側精索靜脈曲張或左側不能平躺消退者 → 懷疑後腹腔腫瘤（如 RCC）壓迫 IVC/腎靜脈**
- 被膜由外到內：皮膚 → dartos → external spermatic fascia → cremasteric fascia → internal spermatic fascia → tunica vaginalis（parietal & visceral）→ tunica albuginea
- **Cremasteric reflex**：L1–L2；輕觸大腿內側 → 同側睪丸上提 → 睪丸扭轉時**消失**

### 1.6 腎臟生理重點

- GFR 正常約 120 mL/min（以 inulin clearance 為 gold standard；臨床用 creatinine clearance 估算）
- Renin-Angiotensin-Aldosterone System（RAAS）：
    - 近腎小球器（JGA）感受低灌流 → 分泌 renin
    - Angiotensin II → 收縮 efferent arteriole > afferent → 維持 GFR（ACEi/ARB 抑制此效應 → GFR 初期下降）
- **Erythropoietin（EPO）**：腎臟分泌（peritubular interstitial cells），缺氧時增加
    - 腎衰竭 → EPO ↓ → normocytic normochromic anemia
- **Vitamin D 活化**：25-(OH)D₃ → 腎臟 1α-hydroxylase → **1,25-(OH)₂D₃**（活性態）
- **MRI 注意事項**：GFR < 30 mL/min 時，含 gadolinium 對比劑可引發 **Nephrogenic Systemic Fibrosis（NSF）**（皮膚及內臟纖維化，無有效治療）→ 嚴重腎功能不全者應避免使用 gadolinium-based contrast

## 二、泌尿道感染（UTI）

### 2.1 分類

- **Uncomplicated UTI**：健康、非懷孕的女性之下泌尿道感染（cystitis）
- **Complicated UTI**：有結構或功能異常（結石、阻塞、導管、男性、孕婦、DM、免疫低下、腎移植）
- **上泌尿道感染**：Acute pyelonephritis
- **Recurrent UTI**：6 個月內 ≥ 2 次 或 1 年內 ≥ 3 次

### 2.2 病原菌

| 排名 | 病原菌 | 備註 |
|------|--------|------|
| 1 | **E. coli** | **最常見**（>75–80% uncomplicated UTI） |
| 2 | Klebsiella | 常見於院內感染 |
| 3 | Proteus mirabilis | 產 urease → 鹼化尿液 → **鳥糞石（struvite）結石** |
| 4 | Enterococcus | Gram (+) |
| 5 | Staphylococcus saprophyticus | **年輕性活躍女性**第二常見 |

- 院內感染/導尿管相關：E. coli 仍最多，但 Pseudomonas、Enterococcus、Candida 比例增加
- **Staphylococcus aureus** 尿路感染 → 需考慮**血行性播散**（菌血症來源）

### 2.3 臨床表現

- **Cystitis（膀胱炎）**：頻尿、急尿、排尿疼痛（dysuria）、恥骨上疼痛、血尿；無發燒
- **Pyelonephritis（腎盂腎炎）**：發燒（>38°C）、畏寒、腰痛（CVA tenderness）、噁心嘔吐 ± 下泌尿道症狀
- 尿液檢查：pyuria（WBC > 5/HPF）、bacteriuria、**leukocyte esterase (+)**、**nitrite (+)**（Gram-negative 菌將 nitrate 轉為 nitrite；Enterococcus 等 Gram(+) 不產生 nitrite → 可能假陰性）

### 2.4 治療

| 情境 | 首選治療 | 療程 |
|------|----------|------|
| Uncomplicated cystitis | **TMP-SMX**（3 天）或 Nitrofurantoin（5 天）或 Fosfomycin（單劑） | 3–5 天 |
| Acute pyelonephritis（輕） | Oral fluoroquinolone（Ciprofloxacin） | 7–14 天 |
| Acute pyelonephritis（重/住院） | IV Ceftriaxone 或 Fluoroquinolone 或 Ampicillin + Gentamicin | 14 天 |
| Complicated UTI | 依培養結果，經驗性先用廣效 | 7–14 天 |
| 孕婦 | Amoxicillin、Cephalosporin、Nitrofurantoin（避免第一孕期及近預產期） | 7 天 |

- **孕婦禁用**：Fluoroquinolone（軟骨毒性）、TMP-SMX（第一孕期葉酸拮抗、第三孕期核黃疸風險）
- **無症狀菌尿（asymptomatic bacteriuria）**：一般不需治療。**例外須治療者**：
    - **孕婦**（→ 30% 進展為 pyelonephritis）
    - 即將接受泌尿道手術者

### 2.5 前列腺炎（Prostatitis）

#### 分類（NIH Classification）

| 類型 | 特點 | 治療 |
|------|------|------|
| **I. Acute bacterial prostatitis** | 發燒、畏寒、會陰/下背痛、排尿症狀；DRE 攝護腺腫脹觸痛（**避免用力按壓 → 菌血症風險**）；常見菌 E. coli | IV 抗生素（Fluoroquinolone 或 TMP-SMX）→ 口服共 4–6 週 |
| **II. Chronic bacterial prostatitis** | 反覆 UTI（同一菌種）、骨盆/會陰不適；EPS 或 post-massage urine 培養陽性 | 口服 Fluoroquinolone 4–6 週（穿透攝護腺佳） |
| **III. Chronic prostatitis / CPPS** | 最常見類型；骨盆疼痛 ≥ 3 個月；培養陰性；分 IIIA（inflammatory）與 IIIB（non-inflammatory） | α-blocker、NSAID、物理治療；無明確特效藥 |
| **IV. Asymptomatic inflammatory** | 切片或精液中偶然發現白血球 | 通常不需治療 |

- Acute bacterial prostatitis 可併發攝護腺膿瘍 → CT/TRUS 確認 → 引流

### 2.6 特殊泌尿道感染

#### 腎膿瘍（Renal / Perinephric Abscess）

- 常見於 DM、結石阻塞
- CT 可見 ring-enhancing lesion
- < 3 cm：抗生素；≥ 3 cm 或無改善 → **經皮引流（percutaneous drainage）**

#### 氣腫性腎盂腎炎（Emphysematous Pyelonephritis）

- 好發 **DM** 患者（90%），最常見菌：E. coli
- CT：腎實質內**氣體**
- 死亡率高 → **積極手術（nephrectomy）** + 抗生素

#### 黃色肉芽腫性腎盂腎炎（Xanthogranulomatous Pyelonephritis, XGP）

- 慢性感染 + 結石阻塞 → 腎臟腫大，被 lipid-laden macrophages 取代
- 常見菌：Proteus、E. coli
- CT：**Bear paw sign**（擴張的腎盞被肉芽腫取代）
- 常被誤為 RCC → 治療：**Nephrectomy**

#### 導尿管相關 UTI（CAUTI）

- 最常見的院內感染
- 預防：儘早移除導管、避免不必要放置
- 治療：換管 + 抗生素（依培養結果）

#### 泌尿道結核

- 血行播散自肺結核
- **無菌性膿尿（sterile pyuria）**：尿液 WBC 多但常規培養陰性 → 高度懷疑
- 影像：腎鈣化、輸尿管狹窄（pipe-stem ureter）、膀胱攣縮（thimble bladder）
- 診斷：尿液 AFB stain/culture（需連續 3 天晨尿）、PCR
- 治療：標準抗結核藥物（RIPE）6–9 個月

## 三、尿路結石（Urolithiasis）

### 3.1 流行病學與風險因子

- 男性 > 女性（約 2–3：1）
- 好發年齡：20–50 歲
- 風險因子：脫水（**最重要可修正因子**）、高蛋白飲食、高鈉飲食、高草酸飲食、家族史、肥胖、痛風、副甲狀腺亢進、反覆 UTI、結構異常
- **終生復發率高達 50%**

### 3.2 結石種類

| 結石 | 佔比 | 特性 | 尿液pH | X光 |
|------|------|------|--------|-----|
| **Calcium oxalate** | **最常見（70–80%）** | 信封形結晶；最常與 hypercalciuria 相關 | 無關 | **Radio-opaque** |
| Calcium phosphate | 常與 CaOx 混合 | — | 鹼性 | Radio-opaque |
| **Struvite（鳥糞石）** | 10–15% | MgNH₄PO₄；urease-producing bacteria（**Proteus**）| **鹼性** | Radio-opaque |
| **Uric acid** | 5–10% | 痛風、高尿酸、腫瘤溶解症候群 | **酸性（<5.5）** | **Radiolucent（X光不顯影！）** |
| Cystine | 1–2% | 體染色體隱性 cystinuria；六角形結晶 | 酸性 | 微弱顯影 |

- **Struvite 結石可形成 staghorn calculus（鹿角形結石）**，填滿整個腎盂腎盞
- Uric acid 結石：X光看不到，但 CT 可以看到 → 唯一可用**鹼化尿液（NaHCO₃ / K-citrate，目標 pH 6.5–7.0）完全溶解**的結石

### 3.3 與高血鈣相關的結石

- **原發性副甲狀腺亢進（Primary hyperparathyroidism）**：
    - 反覆腎結石 → 查 serum Ca²⁺ + intact PTH
    - 高血鈣 + PTH 不適當升高 → 診斷
    - 治療：Parathyroidectomy

### 3.4 臨床表現

- **腎絞痛（Renal colic）**：突發、劇烈的腰部/側腹疼痛，輻射至鼠蹊部/大腿內側
- 疼痛為間歇性，患者坐立不安（與腹膜炎患者不動相反）
- 可伴隨噁心、嘔吐、血尿（>85%有鏡下血尿）
- 結石卡在 UVJ → 類似膀胱刺激症狀（頻尿、急尿）

### 3.5 診斷

- **Non-contrast CT（NCCT）**：**首選**（gold standard），敏感度 >95%
    - 可看到所有類型結石（包括 uric acid），評估水腎程度
- KUB X-ray：可看到 radio-opaque 結石，但敏感度不佳
- 超音波：**孕婦**及小兒首選，可看水腎，但對輸尿管結石敏感度差
- IVP（Intravenous Pyelography）：已少用，被 CT 取代

### 3.6 治療

#### 保守治療（Expectant / Medical Expulsive Therapy, MET）

- 適應症：結石 **< 5 mm** 且症狀可控、無感染/阻塞
- 自然排出率：< 5 mm → **~90%** 可自行排出；5–10 mm → ~50%；>10 mm → 很少
- MET：**α-blocker（Tamsulosin）** → 放鬆輸尿管平滑肌，促進排石
- 充足水分、止痛（NSAID 為首選鎮痛 — Ketorolac；避免 aspirin）

#### 積極治療

| 方法 | 適應症 | 說明 |
|------|--------|------|
| **ESWL（體外震波碎石）** | 腎臟/上段輸尿管結石 **< 2 cm** | 非侵入性；禁忌：**懷孕**、凝血異常、結石後方有動脈瘤、**阻塞遠端未引流** |
| **URS（輸尿管鏡碎石）** | 中下段輸尿管結石；ESWL 失敗 | 可用 Holmium laser 碎石；較 ESWL 成功率高 |
| **PCNL（經皮腎臟碎石）** | 腎結石 **> 2 cm**、staghorn calculus | 經皮穿刺進入腎盂，直接碎石取出 |
| Open / Laparoscopic surgery | 罕用，極複雜或其他方法失敗時 | — |

- **ESWL 禁忌特別記**：**懷孕**、未矯正的凝血障礙、UTI/urosepsis（未控制的感染）、distal obstruction
- **Cystine 及 calcium oxalate monohydrate 結石**對 ESWL 效果較差（硬度高）

### 3.7 結石合併感染 → 泌尿道敗血症

- **最危險的狀況**：阻塞 + 感染（obstructive pyelonephritis）→ 可快速進展為 urosepsis
- 處理：**緊急引流**（PCN 經皮腎造瘻 或 ureteral stent / DJ stent）+ 抗生素 → 待感染控制後再處理結石
- **不可在急性感染期做 ESWL** → 會造成菌血症

### 3.8 預防復發

- 所有患者：**增加水分攝取**（每日尿量 > 2.5 L）、低鈉飲食
- Calcium oxalate：適量鈣質攝取（**不應限鈣**，限鈣反而增加草酸吸收）、低草酸飲食、Thiazide 利尿劑（↓ 尿鈣）、**Potassium citrate**（↑ 尿 **citrate**，抑制結晶形成；**citrate** 與鈣結合形成可溶性複合物，減少 calcium oxalate/phosphate 沉澱）
- Uric acid：鹼化尿液（K-**citrate**）、低普林飲食、Allopurinol
- Struvite：根除感染、完全清除結石（殘餘結石會再感染復發）；Acetohydroxamic acid（urease inhibitor）
- Cystine：大量飲水（> 3 L/day）、鹼化尿液、Tiopronin 或 D-penicillamine（結合 cystine）
- **尿液 citrate 為最重要的結石抑制因子之一**（低尿 citrate = hypocitraturia → 結石風險↑）

## 四、良性攝護腺肥大（BPH）

### 4.1 流行病學與病理

- 極為常見：50 歲男性約 50%，80 歲約 80% 有組織學 BPH
- 好發於 **transition zone**
- 需要 **androgen（DHT）** 才能發生 → 青春期前接受睪丸切除者不會發生 BPH
- **5α-reductase** 將 testosterone 轉為 **DHT**（更強效的 androgen）

### 4.2 症狀（LUTS = Lower Urinary Tract Symptoms）

#### 儲尿症狀（Storage / Irritative）
- 頻尿（frequency）、夜尿（nocturia）、急尿（urgency）、急迫性尿失禁

#### 排尿症狀（Voiding / Obstructive）
- 尿流變細/無力（weak stream）、用力排尿（straining）、尿分叉、排尿中斷（intermittency）、排尿遲疑（hesitancy）、尿後滴瀝（post-void dribbling）

- 嚴重度評估：**IPSS（International Prostate Symptom Score）**（0–35 分）
    - Mild：0–7；Moderate：8–19；Severe：20–35
- **注意：LUTS 症狀嚴重度與指診時攝護腺大小沒有絕對相關性**（小攝護腺可有嚴重症狀，大攝護腺可能無症狀）

### 4.3 併發症

- 急性尿滯留（Acute urinary retention, AUR）
- 反覆 UTI
- 膀胱結石
- 肉眼血尿
- 腎水腫（bilateral hydronephrosis）→ 腎後性腎衰竭
- 膀胱逼尿肌失代償（detrusor failure）

### 4.4 診斷

- **DRE（digital rectal exam）**：攝護腺對稱性腫大、質地如橡膠、表面光滑（與癌鑑別：硬、不規則、結節）
- PSA（前列腺特異抗原）：BPH 也會升高，但通常 < 10 ng/mL
- 尿流速（Uroflowmetry）：Qmax < 10 mL/s 提示明顯阻塞
- 殘餘尿量（Post-Void Residual）：> 100–200 mL 為異常
- 超音波：評估攝護腺大小、殘尿、上泌尿道

### 4.5 治療

#### 藥物治療

| 藥物 | 機制 | 效果 | 副作用 |
|------|------|------|--------|
| **α₁-blocker**（Tamsulosin, Doxazosin） | 放鬆攝護腺/膀胱頸平滑肌 | 快速改善症狀（數天） | 姿態性低血壓、逆行性射精、**Intraoperative Floppy Iris Syndrome（IFIS）**→ 白內障手術需告知 |
| **5α-reductase inhibitor**（Finasteride, Dutasteride） | 抑制 DHT 合成 → 攝護腺縮小 | 需 3–6 個月見效；減少 AUR 及手術風險 | 性慾下降、勃起功能障礙、**PSA 降低約 50%**（→ PSA 需 ×2 校正） |
| PDE5 inhibitor（Tadalafil） | cGMP↑ → 平滑肌放鬆 | 同時改善 LUTS 與 ED | — |
| 抗膽鹼藥（Anticholinergics） | 抑制逼尿肌 | 針對儲尿症狀（OAB） | 口乾、便秘、尿滯留風險 |
| β₃-agonist（Mirabegron） | 膀胱放鬆 | 儲尿症狀，尿滯留風險較低 | 高血壓 |

#### 手術治療

- **適應症**：藥物無效、AUR、反覆 UTI、膀胱結石、腎水腫/腎衰竭、反覆血尿
- **TURP（經尿道攝護腺切除術）**：**Gold standard**
    - 適用攝護腺 30–80 g
    - 併發症：
        - **TURP syndrome（低血鈉稀釋性低鈉血症）**：灌洗液（低張液如 glycine/sorbitol）大量吸收 → **dilutional hyponatremia** → 意識混亂、噁心、視力模糊、抽搐、心律不整
        - 預防：使用 bipolar TURP（可用生理食鹽水灌洗）；縮短手術時間（<60–90 分鐘）
        - 治療：停止手術、利尿劑、必要時 hypertonic saline
        - 出血、尿道狹窄、逆行性射精（最常見長期併發症，50–75%）、尿失禁（罕見）
- **Open prostatectomy**：攝護腺 >80–100 g
- **HoLEP（Holmium Laser Enucleation）**：不限大小，出血少，需要經驗
- **其他微創**：UroLift、Aquablation、Rezūm（水蒸氣治療）

## 五、攝護腺癌（Prostate Cancer）

### 5.1 流行病學

- 男性最常見的癌症之一（歐美第一，台灣亦逐年上升）
- 好發 > 50 歲，年齡是最大風險因子
- 風險因子：年齡、家族史、非裔、高脂飲食
- 多數發生在 **peripheral zone**（70%）

### 5.2 病理

- **>95% 是 adenocarcinoma（腺癌）**
- **Gleason grading system**：
    - 取最多的 pattern + 第二多的 pattern → Gleason score（2–10）
    - 現已改用 **Grade Group（ISUP）**：
        - Grade Group 1：Gleason ≤ 6（低風險）
        - Grade Group 2：Gleason 3+4=7
        - Grade Group 3：Gleason 4+3=7
        - Grade Group 4：Gleason 8
        - Grade Group 5：Gleason 9–10（高風險）
- 攝護腺癌常為 **multifocal**

### 5.3 篩檢與診斷

- **PSA（Prostate-Specific Antigen）**：
    - 正常 < 4 ng/mL（但無絕對 cutoff）
    - PSA 是 organ-specific，**不是 cancer-specific** → BPH、prostatitis、騎車、射精、DRE 也會升高
    - **Free PSA / Total PSA ratio**：癌症時 free PSA 比例較低（< 25% 懷疑癌）
    - **PSA density**：PSA / 攝護腺體積 → > 0.15 懷疑癌
    - **PSA velocity**：> 0.75 ng/mL/year 提示癌
- **DRE**：硬結節、不規則、不對稱
- **確診**：**經直腸超音波引導切片（TRUS biopsy）** 或 MRI-fusion targeted biopsy
    - **mpMRI（multiparametric MRI）**：PI-RADS 評分 → ≥ 3 建議切片

### 5.4 分期（TNM）

- **T1**：臨床未摸到/影像未見（incidental，TURP 或切片發現）
- **T2**：侷限於攝護腺內（T2a: ≤ 半葉; T2b: > 半葉; T2c: 雙葉）
- **T3**：穿出攝護腺被膜（T3a）或侵犯精囊（T3b）
- **T4**：侵犯周圍結構（膀胱頸、外括約肌、直腸、骨盆壁）
- 轉移：**骨轉移最常見**（osteoblastic / 成骨性 → bone scan 呈 hot spots）
    - 與其他癌症骨轉移多為 osteolytic 不同
- 骨盆腔淋巴結 → 遠處淋巴結 → 骨（axial skeleton, 如腰椎骨盆）

### 5.5 治療

#### 局限性攝護腺癌（Localized）

| 風險分層 | 選項 |
|----------|------|
| 極低/低風險（GG1, PSA<10, ≤T2a） | **Active surveillance**（定期 PSA、DRE、切片追蹤） |
| 中風險 | Radical prostatectomy（RP）或 Radiation therapy（RT） |
| 高風險 | RP 或 RT + ADT（Androgen Deprivation Therapy） |

- **Radical prostatectomy** 併發症：勃起功能障礙（**cavernous nerve 受損**；nerve-sparing 技術可降低）、尿失禁
- **Radiation therapy**（EBRT 或 brachytherapy）：併發症包括 radiation cystitis/proctitis

#### 進展性/轉移性攝護腺癌

- **Androgen Deprivation Therapy（ADT / 去雄性素治療）**為主要治療
    - **GnRH agonist（Leuprolide）**：初期會有 testosterone flare → 需合用 anti-androgen 預防 flare（Bicalutamide）
    - **GnRH antagonist（Degarelix）**：無 flare 現象
    - 雙側睪丸切除（bilateral orchiectomy）：最快達到去勢水準
    - 去勢水準：testosterone < 50 ng/dL
- ADT 副作用：熱潮紅、性慾降低、勃起障礙、骨質疏鬆、肌肉流失、代謝症候群、心血管風險增加
- **CRPC（Castration-Resistant Prostate Cancer）**：去勢後仍進展
    - 二線荷爾蒙治療：**Abiraterone**（CYP17 抑制劑）+ prednisone、**Enzalutamide**（AR antagonist）
    - 化療：**Docetaxel**（首選）、Cabazitaxel
    - 其他：Radium-223（骨轉移）、Sipuleucel-T（免疫治療）

## 六、膀胱癌與上泌尿道上皮癌

### 6.1 流行病學與風險因子

- 泌尿道最常見的惡性腫瘤
- 男性 > 女性（3–4：1）
- 好發 > 60 歲
- **最重要風險因子：吸菸（#1）**
- 其他風險因子：
    - **芳香胺類（aromatic amines）**：2-naphthylamine → 橡膠、染料、化工業
    - **Cyclophosphamide**（出血性膀胱炎 → acrolein 代謝物）→ 預防用 **Mesna**
    - 慢性膀胱炎/結石刺激 → squamous cell carcinoma
    - **Schistosoma haematobium（埃及血吸蟲）** → **squamous cell carcinoma**
    - 放射線治療後

### 6.2 病理

| 類型 | 佔比 | 關聯 |
|------|------|------|
| **Urothelial carcinoma（transitional cell）** | **>90%** | 吸菸、芳香胺 |
| Squamous cell carcinoma | 3–5% | 血吸蟲、慢性刺激 |
| Adenocarcinoma | 1–2% | Urachal remnant（臍尿管癌，位於 dome） |

- Urothelial carcinoma 重要特性：**field change effect** → 整條泌尿上皮（腎盂→輸尿管→膀胱→尿道）都有致癌風險 → 上泌尿道 urothelial carcinoma 術後需追蹤膀胱

### 6.3 臨床表現

- **無痛性肉眼血尿（painless gross hematuria）**：**最常見且最重要的表現**
- 任何 > 40 歲之無痛性血尿 → 需排除膀胱癌
- 可能有頻尿、急尿等刺激症狀（尤其 CIS）
- 進展期：骨盆疼痛、水腎、體重下降

### 6.4 診斷

- **Cystoscopy（膀胱鏡）+ biopsy**：**Gold standard**
- 尿液細胞學（Urine cytology）：高 grade 腫瘤及 CIS 敏感度高，低 grade 敏感度差
- CT urography：評估上泌尿道，排除同步上泌尿道腫瘤
- **TURBT（經尿道膀胱腫瘤切除術）**：既是診斷也是初步治療 → 需切到肌肉層以判斷有無 muscle invasion

### 6.5 分期

- **Non-muscle invasive（NMIBC）**：
    - **Ta**：乳突狀，侷限於黏膜（最常見）
    - **T1**：侵犯 lamina propria（黏膜下層）
    - **CIS（Tis, Carcinoma in situ）**：**flat, high-grade** → 外觀為紅色天鵝絨斑塊 → 高度惡性，侵犯風險高
- **Muscle-invasive（MIBC）**：
    - **T2**：侵犯肌肉層（detrusor muscle）
    - **T3**：穿出膀胱壁至周圍脂肪
    - **T4**：侵犯鄰近器官（攝護腺、子宮、骨盆壁）

### 6.6 治療

#### Non-muscle invasive bladder cancer（NMIBC）

- **TURBT**（完全切除腫瘤）
- 術後依風險分層：
    - 低風險（low-grade Ta, 單發小腫瘤）：TURBT + 單劑即刻膀胱灌注化療（Mitomycin C，術後 24 小時內）
    - 中風險：TURBT + 膀胱灌注化療（Mitomycin C 或 BCG）
    - 高風險（T1、high-grade、CIS、多發/大腫瘤）：TURBT + **BCG 膀胱灌注**（首選）
- **BCG（Bacillus Calmette-Guérin）**：
    - 減毒活菌卡介苗 → 激發局部免疫反應
    - **CIS 的首選治療**
    - 副作用：膀胱刺激症狀、血尿、BCG 感染/sepsis（罕見但嚴重 → INH + Rifampin 治療）
    - **禁忌**：免疫抑制、活動性 UTI、肉眼血尿、創傷性導管插入
- BCG 失敗 → 考慮 **radical cystectomy**

#### Muscle-invasive bladder cancer（MIBC）

- **Radical cystectomy + pelvic lymph node dissection** 為 gold standard
    - 男性：膀胱 + 攝護腺 + 精囊切除
    - 女性：膀胱 + 子宮 + 卵巢 + 前陰道壁切除
    - 尿路重建：**Ileal conduit（迴腸通道）**最常用；orthotopic neobladder（新膀胱）
- **Neoadjuvant cisplatin-based chemotherapy** → 術前化療可改善存活
- 不適合手術者 → chemoradiation（膀胱保留策略）

### 6.7 上泌尿道上皮癌（Upper Tract Urothelial Carcinoma, UTUC）

- 佔 urothelial carcinoma 的 5–10%
- 風險因子：同膀胱癌（吸菸、芳香胺），加上 **Balkan nephropathy、aristolochic acid（馬兜鈴酸）**
    - **台灣因中草藥使用較多，UTUC 比例世界偏高**
- 臨床：血尿、側腰痛
- 診斷：CT urography、ureteroscopy + biopsy、urine cytology
- 治療：**Radical nephroureterectomy + bladder cuff excision**（標準術式）
    - 術後需定期膀胱鏡追蹤（field change effect → 膀胱復發率 20–50%）

## 七、腎細胞癌（Renal Cell Carcinoma, RCC）

### 7.1 流行病學

- 成人腎臟最常見的惡性腫瘤（佔 85–90%）
- 男性 > 女性（2：1）
- 好發 50–70 歲
- 風險因子：吸菸（#1 modifiable）、肥胖、高血壓、VHL syndrome、長期透析（acquired cystic disease）

### 7.2 病理分類

| 亞型 | 佔比 | 特點 |
|------|------|------|
| **Clear cell** | **70–80%** | 最常見；與 **VHL gene（3p deletion）** 相關；富含脂質/糖原 → 細胞透明 |
| Papillary | 10–15% | Type 1（MET mutation）、Type 2；雙側/多發較多 |
| Chromophobe | 5% | 預後較好 |
| Collecting duct | <1% | 罕見、預後差 |
| Oncocytoma | — | **良性腫瘤**，嗜酸性細胞，central scar on CT |

### 7.3 臨床表現

- **經典三徵（Classic triad）**：血尿 + 腰痛 + 腹部腫塊 → 但只有 **10–15%** 有完整三徵（出現時通常已晚期）
- 現代大多為影像**偶然發現**（incidental finding）
- **Paraneoplastic syndromes**（RCC 被稱為 "internist's tumor"）：
    - **高血鈣**（PTHrP）
    - **紅血球增多症（Erythrocytosis）**：EPO 分泌
    - **Stauffer syndrome**：非轉移性肝功能異常（hepatic dysfunction without metastasis）→ 腎切除後可恢復
    - 高血壓（Renin 分泌）
    - Cushing syndrome（ACTH）
- **左側精索靜脈曲張**（不能平躺消退）→ 左腎靜脈被腫瘤侵犯/壓迫
- 轉移：肺（最常見）、骨、肝、腦
    - RCC 有特殊傾向沿 **renal vein → IVC** 生長形成 tumor thrombus（甚至延伸至右心房）

### 7.4 診斷

- **CT with contrast（三期 CT）**：首選影像
    - 典型 clear cell RCC：**明顯強化（enhancement > 15 HU）**
    - Bosniak classification（腎臟囊性病灶分類）：
        - I：單純囊腫 → 觀察
        - II：少量鈣化/薄隔 → 觀察
        - IIF：更多隔膜/鈣化 → 追蹤
        - III：厚壁/隔膜、強化 → **手術或切片**（50% malignancy）
        - IV：明顯實質成分、強化 → **手術**（>90% malignancy）
- MRI：評估 IVC tumor thrombus 最佳
- **不需要術前切片**（典型影像即可手術）→ 切片有出血、needle tract seeding 風險

### 7.5 治療

#### 局限性 RCC

| 情境 | 治療 |
|------|------|
| T1a（≤ 4 cm） | **Partial nephrectomy（腎部分切除）**首選（保留腎功能） |
| T1b（4–7 cm） | Partial 或 Radical nephrectomy |
| T2（> 7 cm） | **Radical nephrectomy** |
| T3–T4 | Radical nephrectomy ± IVC thrombectomy |
| 小腫瘤、年老/合併症多 | Active surveillance 或 ablation（RFA/cryoablation） |

- Radical nephrectomy 包括：腎臟 + Gerota's fascia + 腎上腺（若上極腫瘤或大腫瘤）+ 區域淋巴結
- RCC 對傳統化療及放療 **不敏感**

#### 轉移性 RCC

- **標靶治療為主**：
    - **TKI（Tyrosine Kinase Inhibitor）**：Sunitinib、Pazopanib、Cabozantinib
    - **mTOR inhibitor**：Everolimus、Temsirolimus
    - **Anti-VEGF**：Bevacizumab
- **免疫治療**：
    - **Checkpoint inhibitors**：Nivolumab（anti-PD-1）、Ipilimumab（anti-CTLA-4）→ 目前一線組合治療
    - **IFN-α（Interferon-alpha）**：早期使用的免疫療法，單獨效果有限，現多被 checkpoint inhibitors 取代
    - 高劑量 IL-2（historical，少數可長期完全反應）
- **Cytoreductive nephrectomy**：轉移性但原發腫瘤仍在 → 某些情況切除原發可改善全身治療效果

## 八、睪丸癌（Testicular Cancer）

### 8.1 流行病學

- **15–35 歲男性最常見的惡性腫瘤**
- 風險因子：**隱睪症（cryptorchidism）**（最重要，即使手術矯正仍有較高風險）、對側睪丸癌史、家族史、Klinefelter syndrome、testicular microlithiasis
- 隱睪症：最常停留在**腹股溝管**；惡性化風險最高的是**腹腔內**未降睪丸
    - 最常見的隱睪症相關癌：**Seminoma**

### 8.2 分類

| 大類 | 亞型 | 佔比 | 特點 |
|------|------|------|------|
| **Germ Cell Tumor（GCT）** | — | **>95%** | — |
| GCT - Seminoma | **Seminoma** | 40–50% | 最常見單一類型；對放療極度敏感；分泌 **β-hCG**（10%）但 **AFP 永遠正常** |
| GCT - Non-seminoma | **Embryonal carcinoma** | — | Non-seminoma 中分化最差的成分；常混合於 mixed GCT 中 |
| | **Yolk sac tumor（Endodermal sinus）** | — | 小兒最常見 GCT；分泌 **AFP**；Schiller-Duval bodies |
| | **Choriocarcinoma** | — | 分泌 **β-hCG**；早期血行轉移（肺、腦）；預後最差（因早期廣泛轉移） |
| | **Teratoma** | — | 成熟/不成熟；化療後殘留常為 teratoma → 需手術切除 |
| | **Mixed GCT** | 40% | 含 ≥2 種以上成分 |
| **Non-GCT（性索-間質）** | Leydig cell tumor | — | 分泌 testosterone/estrogen → 男童性早熟/男性女乳症；多為良性 |
| | Sertoli cell tumor | — | 多為良性 |
| **Lymphoma** | — | — | **>60 歲睪丸腫瘤最常見為 lymphoma**（多為 DLBCL）；常雙側 |

### 8.3 腫瘤標記

| 標記 | Seminoma | Non-seminoma |
|------|----------|--------------|
| **AFP** | **永遠正常**（AFP 升高 → 排除 pure seminoma） | ↑（yolk sac tumor、embryonal） |
| **β-hCG** | 可輕微升高（10%） | ↑（choriocarcinoma、embryonal） |
| **LDH** | 升高與腫瘤體積相關 | 升高 |

- **任何「seminoma」病理但 AFP 升高 → 視為 non-seminoma 治療**

### 8.4 臨床表現

- **無痛性睪丸腫大**：最典型表現
- 沉重感、陰囊不適
- 超音波：實質性、hypoechoic、血流增加
- 重要鑑別：**透光試驗（transillumination）陰性**（實質腫瘤不透光；hydrocele 透光）
- 10% 以轉移症狀表現（腰痛—後腹腔淋巴結、咳嗽—肺轉移）
- 睪丸癌**不會引起陰囊痛的急症**（需與 torsion 鑑別）

### 8.5 診斷與分期

- 超音波：首選影像 → 可區分 intratesticular（高度懷疑惡性）vs extratesticular 病灶
- 腫瘤標記：AFP、β-hCG、LDH
- **確診手術：Radical inguinal orchiectomy**（經腹股溝切除，**非經陰囊！** → 避免改變淋巴引流路徑）
    - **禁止經陰囊切片（transscrotal biopsy）** → 改變淋巴引流至腹股溝淋巴結
- 分期影像：胸部/腹部/骨盆 CT
- 睪丸的淋巴引流：→ **後腹腔淋巴結（retroperitoneal/para-aortic lymph nodes）**（非腹股溝！因為胚胎來源在後腹腔）
    - 右側 → interaortocaval LN
    - 左側 → para-aortic LN

### 8.6 治療

#### Seminoma

| Stage | 治療 |
|-------|------|
| I（侷限於睪丸） | Orchiectomy + **surveillance**（首選）或 adjuvant radiation / single carboplatin |
| IIA/B（retroperitoneal LN ≤ 5 cm） | Orchiectomy + **Radiation therapy**（對放療極度敏感） |
| IIC 以上/轉移 | **Cisplatin-based chemotherapy**（BEP：Bleomycin + Etoposide + Cisplatin） |

#### Non-seminoma

| Stage | 治療 |
|-------|------|
| I | Orchiectomy + surveillance 或 RPLND 或 adjuvant chemo（1–2 cycles BEP） |
| II 以上/轉移 | **BEP chemotherapy**（3–4 cycles）→ 殘餘腫塊 → **RPLND（retroperitoneal lymph node dissection）** |

- **RPLND** 後最重要的併發症：**逆行性射精**（損傷交感神經鏈）→ nerve-sparing 技術可降低
- **BEP 副作用**：Bleomycin → **肺纖維化**（cumulative dose-related）；Cisplatin → 腎毒性、耳毒性
- 睪丸癌預後極佳：整體 cure rate > 95%；即使轉移性 seminoma 5 年存活率也 > 90%

## 九、泌尿道創傷

### 9.1 腎臟創傷

- 泌尿道最常受傷的器官（GU trauma 中佔 80%）
- 鈍傷（80–90%）> 穿透傷

#### AAST 分級

| Grade | 損傷程度 |
|-------|----------|
| I | 皮下血腫（subcapsular）、無擴張；或皮質裂傷 < 1 cm、不到 collecting system |
| II | 血腫未擴張（perirenal）；皮質裂傷 < 1 cm、無尿外漏 |
| III | 皮質裂傷 > 1 cm、無尿外漏 |
| IV | 穿入 collecting system（有尿外漏）、或腎動靜脈損傷有 contained hemorrhage |
| V | 腎臟碎裂（shattered kidney）、或腎蒂血管斷裂（devascularized） |

#### 處理原則

- **大多數（85%）可非手術治療**（包括 Grade I–IV 穩定者）
- 血行不穩定 → 先 FAST/CT → 必要時手術
- 穿透傷（尤其槍傷）→ 較多需手術探查
- **手術指徵**：持續出血血行不穩、Grade V、expanding/pulsatile perirenal hematoma（術中發現）、renal pedicle avulsion
- 術前確認對側腎功能正常（one-shot IVP 或 CT）

### 9.2 輸尿管創傷

- 罕見（佔 GU trauma < 1%），多為 **醫源性（iatrogenic）**
    - 常見於：婦科手術（子宮切除）、大腸手術、URS
- 醫源性損傷最常發生位置：**骨盆段**（靠近子宮動脈 / 靠近 UVJ）
- 診斷：術中直視、CT with delayed phase（看 contrast extravasation）、retrograde pyelography
- 治療：
    - 上段：ureteroureterostomy（端端吻合）
    - 中段：ureteroureterostomy、transureteroureterostomy
    - 下段：**ureteroneocystostomy（輸尿管再植入膀胱）** ± Boari flap ± Psoas hitch
    - 完全缺損很長：ileal ureter、autotransplantation、nephrectomy

### 9.3 膀胱創傷

- 常與 **骨盆骨折** 相關
- 分類與處理：

| 類型 | 機制 | 位置 | 治療 |
|------|------|------|------|
| **Extraperitoneal rupture** | 骨盆骨折碎片穿刺 | 前壁/側壁 | **Foley catheter 引流 10–14 天**（保守治療） |
| **Intraperitoneal rupture** | 腹部鈍傷時膀胱脹滿被壓 | **Dome（穹窿）** | **手術修補** |

- 診斷：**CT cystography** 或 conventional cystography（retrograde instill contrast 350 mL → 看外漏）
- 膀胱裝滿 contrast 後需拍**排空後**影像 → 小的後壁穿孔可能被脹滿的膀胱遮住

### 9.4 尿道創傷

- **男性遠多於女性**（男性尿道長 20 cm；女性僅 4 cm）
- 分為 anterior（bulbar, pendulous）與 posterior（prostatic, membranous）

| 類型 | 常見機制 | 特點 |
|------|----------|------|
| **Posterior urethral injury** | **骨盆骨折**（straddle-type forces） | 膜部尿道斷裂；血尿、無法排尿、**high-riding prostate on DRE** |
| **Anterior urethral injury** | **跨坐傷（straddle injury）**、直接外傷 | Bulbar urethra 最常見；perineal hematoma（butterfly pattern） |

- **重要臨床線索**（懷疑尿道損傷 → 禁放 Foley）：
    1. **Blood at urethral meatus**（尿道口出血）
    2. **High-riding / non-palpable prostate**（DRE）
    3. **Perineal hematoma / ecchymosis**
    4. 無法自解小便

- **診斷**：**Retrograde urethrography（RUG）** → 若有 extravasation → 確認損傷
- **處理**：
    - 後尿道損傷：**Suprapubic cystostomy**（恥骨上膀胱造瘻）→ 待穩定後延遲修補（3–6 個月）
    - 前尿道部分損傷：Foley catheter 引流 或 suprapubic cystostomy
    - 前尿道完全斷裂：手術修補

### 9.5 生殖器創傷

- **睪丸破裂（Testicular rupture）**：超音波確認 tunica albuginea 破裂 → **手術探查修補**（保留睪丸）
- **陰莖骨折（Penile fracture）**：
    - 勃起時外力 → tunica albuginea 斷裂
    - 聽到「啪」一聲 + 立即消腫 + 陰莖彎曲/茄子樣血腫（eggplant deformity）
    - 診斷：臨床即可（超音波/MRI 可輔助）
    - 治療：**緊急手術修補**

## 十、排尿障礙與尿失禁

### 10.1 尿動力學（Urodynamics）基本概念

- **Uroflowmetry（尿流速測量）**：非侵入性
    - Qmax（最大尿流速）：正常 > 15 mL/s；< 10 mL/s 提示阻塞
    - Bell-shaped curve：正常
- **Cystometry（膀胱壓力測量）**：評估儲尿期
    - 正常：膀胱充盈時壓力微升（高順應性 compliance）、無不自主收縮
    - **Detrusor overactivity（DO）**：充盈時出現不自主逼尿肌收縮 → 急尿/急迫性尿失禁
    - **Detrusor underactivity**：收縮力不足 → 排尿困難/大量殘尿
- **Pressure-Flow Study**：排尿期，區分阻塞 vs 逼尿肌無力
    - 高壓力 + 低流速 → 膀胱出口阻塞（Bladder outlet obstruction）
    - 低壓力 + 低流速 → Detrusor underactivity

### 10.2 神經性膀胱（Neurogenic Bladder）

| 損傷位置 | 膀胱行為 | 特點 |
|----------|----------|------|
| **UMN lesion**（suprasacral：SCI above S2、MS、CVA） | **Detrusor overactivity（spastic bladder）** | 不自主收縮、高壓、incontinence；可合併 DSD |
| **LMN lesion**（sacral/infrasacral：S2–S4 損傷、cauda equina、DM neuropathy） | **Detrusor underactivity（flaccid/areflexic bladder）** | 無收縮、大量殘尿、overflow incontinence |

- **DSD（Detrusor-Sphincter Dyssynergia）**：逼尿肌收縮時外括約肌不放鬆（UMN lesion）→ 膀胱內壓極高 → **最危險→ 腎臟損害**
- 治療：**Clean Intermittent Catheterization（CIC）** 為 neurogenic bladder 的 cornerstone management
    - UMN + DO：anticholinergics + CIC
    - UMN + DSD：CIC ± botulinum toxin sphincter injection ± α-blocker
    - LMN：CIC（主要）± Credé maneuver / Valsalva（需小心）

#### Autonomic Dysreflexia（自主神經反射異常）

- 好發於 **T6 以上脊髓損傷**（SCI）患者
- 觸發因素：膀胱過度膨脹（**最常見**）、糞便嵌塞、導尿管阻塞、皮膚刺激
- 機制：損傷部位以下的交感反射亢進，但腦幹下行抑制訊號無法通過 → 損傷部位以下血管收縮
- 症狀：**突發嚴重高血壓**（SBP 可 > 200 mmHg）、劇烈頭痛、反射性心搏過緩、損傷部位以上潮紅/流汗、損傷部位以下蒼白/雞皮疙瘩
- **為急症！** 嚴重可致顱內出血、癲癇、死亡
- 處理：**立即找出並移除觸發因素**（最重要 → 先引流膀胱/解除糞便嵌塞）→ 坐起以降低血壓 → 若仍高血壓 → 短效降壓藥（Nifedipine、Nitropaste）

### 10.3 尿失禁

| 類型 | 機制 | 特點 | 治療 |
|------|------|------|------|
| **Stress（應力性）** | 腹壓增加（咳嗽、打噴嚏）時尿液漏出；括約肌/骨盆底弱 | 女性多見；多產、肥胖、停經 | 骨盆底運動（**Kegel**）、Mid-urethral sling（TVT/TOT）、Burch colposuspension |
| **Urge（急迫性）** | 逼尿肌不自主收縮 → 急尿且來不及上廁所 | OAB 的一種表現 | **Anticholinergics**（Oxybutynin, Tolterodine）、β₃-agonist（Mirabegron）、Botox（onabotulinumtoxinA）注射膀胱 |
| **Overflow（溢流性）** | 膀胱過度膨脹 → 被動漏尿 | 膀胱出口阻塞或 detrusor failure；常有大量排尿後殘餘尿量 | 治療根本原因（BPH → α-blocker/TURP）；CIC |
| **Functional** | 行動/認知障礙無法及時到達廁所 | 老年人 | 環境調整、定時排尿 |
| **Mixed** | Stress + Urge | 常見 | 合併治療 |

### 10.4 膀胱過動症（OAB, Overactive Bladder）

- 定義：急尿（urgency）為核心症狀，可伴隨頻尿、夜尿 ± 急迫性尿失禁
- **是臨床症候群，非尿動力學診斷**（尿動力學上對應的是 detrusor overactivity）
- 治療階梯：行為治療（膀胱訓練、定時排尿）→ 藥物（anticholinergics / Mirabegron）→ Botox → sacral neuromodulation → augmentation cystoplasty

### 10.5 間質性膀胱炎（Interstitial Cystitis / Bladder Pain Syndrome, IC/BPS）

- 慢性（> 6 週）膀胱疼痛/不適，伴頻尿、急尿，**尿液培養陰性**（排除感染）
- 好發中年女性
- 特徵：膀胱充盈時恥骨上疼痛加劇，排尿後緩解
- 膀胱鏡：**Hunner's ulcer**（classic type，約 10%）或 **glomerulations**（膀胱水擴張後黏膜點狀出血）
- 診斷為排除性診斷（排除 UTI、膀胱癌、OAB、結核等）
- 治療：避免刺激性食物（咖啡、酒精、辛辣）→ 口服 **Amitriptyline**（首選口服藥）/ Pentosan polysulfate（Elmiron）→ 膀胱灌注（DMSO、hyaluronic acid）→ 膀胱水擴張 → refractory → sacral neuromodulation / cystectomy（極少數）

## 十一、泌尿系統先天異常

### 11.1 腎臟異常

- **Horseshoe kidney（馬蹄腎）**：
    - 最常見腎臟融合異常
    - 下極融合，被 **IMA（inferior mesenteric artery）** 擋住無法正常上升 → 位置偏低
    - 常無症狀，但易併 **UPJ obstruction** 及結石
    - 與 **Turner syndrome（45,X）** 相關

- **Ectopic kidney（異位腎）**：腎臟位於骨盆腔或其他異常位置
- **Renal agenesis（腎發育不全）**：
    - 雙側 → **Potter sequence**（羊水過少 → 肺發育不全 + 肢體畸形 + 特殊面容）→ 致死
    - 單側 → 通常無症狀，對側代償性肥大

### 11.2 Multicystic Dysplastic Kidney（MCDK）

- 新生兒最常見的囊性腎臟疾病
- 腎臟為多發囊腫取代，無功能
- 多單側、多自行萎縮
- 與 UPJ obstruction 鑑別：MCDK 腎臟無功能（核醫無攝取），UPJ 有功能但延遲排泄

### 11.3 多囊腎（Polycystic Kidney Disease, PKD）

| 特徵 | ADPKD | ARPKD |
|------|-------|-------|
| 遺傳 | **體染色體顯性** | 體染色體隱性 |
| 基因 | **PKD1（Chr 16，85%）**、PKD2 | PKHD1 |
| 發病年齡 | 成人（30–50 歲） | 嬰兒/新生兒 |
| 腎臟外表現 | **肝囊腫**（最常見）、**腦動脈瘤（berry aneurysm）**、心臟瓣膜異常 | 肝纖維化（congenital hepatic fibrosis） |
| 預後 | 進展至 ESRD（60 歲約 50%） | 嚴重者出生即死亡 |

- ADPKD：腎臟雙側腫大、多發囊腫、血尿、腰痛、高血壓、結石
- 治療：控制血壓（ACEi/ARB）、Tolvaptan（V2 receptor antagonist，延緩 ADPKD 進展）

### 11.4 輸尿管異常

- **UPJ obstruction（腎盂輸尿管交界阻塞）**：
    - **新生兒最常見的泌尿道阻塞原因**
    - 產前超音波：水腎（hydronephrosis）
    - 核醫腎臟掃描（MAG3 / DTPA + Lasix renogram）確認阻塞
    - 治療：**Pyeloplasty**（Anderson-Hynes dismembered pyeloplasty）→ 成功率 > 95%

- **VUR（Vesicoureteral Reflux, 膀胱輸尿管逆流）**：
    - 小兒反覆 UTI 最常見的結構異常
    - 正常防逆流機制：輸尿管以斜角穿入膀胱壁（submucosal tunnel），膀胱充盈時壓迫輸尿管形成活瓣；VUR 多因 tunnel 過短所致（原發性）
    - 診斷：**VCUG（Voiding Cystourethrography）** 為 gold standard
    - **分級（International Reflux Grading）**：
        - **I**：逆流僅至輸尿管，未達腎盂
        - **II**：逆流至腎盂腎盞，但無擴張
        - **III**：輸尿管輕度擴張，腎盞輕微鈍化
        - **IV**：輸尿管中度擴張迂曲，腎盞鈍化明顯
        - **V**：輸尿管嚴重擴張迂曲，腎盞杯狀結構消失（massive reflux）
    - **大多會隨年齡自然消退**（低 grade I–III，消退率 > 80%）；高 grade 自然消退率低
    - 處理：
        - Grade I–III：低劑量預防性抗生素（如 TMP-SMX）+ 定期追蹤（VCUG / 超音波 / DMSA 評估腎臟疤痕）
        - Grade IV–V 或 breakthrough UTI / 腎臟疤痕進展 → **Ureteral reimplantation（Cohen 或 Politano-Leadbetter 手術）** 或 endoscopic injection（Deflux）
    - **Weigert-Meyer law**（適用於 complete duplex collecting system）：
        - **上極輸尿管**開口位於膀胱**較內下方**（ectopic）→ 常伴 **ureterocele** 或 **ectopic insertion** → 易發生**阻塞**
        - **下極輸尿管**開口位於膀胱**較外上方**（正常位置偏外側、tunnel 較短）→ 易發生 **VUR**
        - 口訣：上極阻塞、下極逆流

- **Posterior Urethral Valves（PUV，後尿道瓣膜）**：
    - **男嬰最常見的下泌尿道阻塞原因**
    - 產前：雙側水腎 + 膀胱擴大 + oligohydramnios → 可致肺發育不全
    - VCUG：膀胱壁小梁化（trabeculated）+ 後尿道擴張
    - 治療：穩定後 → **Transurethral valve ablation（內視鏡瓣膜切除）**；新生兒期先 vesicostomy

- **Ureterocele**：輸尿管末端囊狀擴張；常與 duplicated collecting system 的上極 ectopic ureter 相關（見 Weigert-Meyer law）
- **Ectopic ureter**：女童反覆漏尿（dripping incontinence）→ 輸尿管開口在括約肌以下

### 11.5 隱睪症（Cryptorchidism）

- 定義：睪丸未降至陰囊內正常位置
- 最常見的男嬰生殖器異常
- 足月嬰兒 3–4%，早產兒高達 30%
- 大部分在 **6 個月內自行下降**
- 若 6 個月仍未下降 → 建議 **6–12 個月大時行 orchiopexy（睪丸固定術）**
    - 降低（但不消除）惡性風險、改善生育力
    - 未治療的風險：不孕、睪丸癌、扭轉、腹股溝疝氣
- 不可觸及的隱睪 → 腹腔鏡探查

### 11.6 尿道下裂（Hypospadias）

- 尿道開口位於陰莖腹側（非正常龜頭頂端）
- 常合併 ventral chordee（陰莖向下彎曲）、prepuce 背側多餘（hooded prepuce）
- **禁做包皮環切術（circumcision）** → 包皮需保留用於後續手術修補
- 治療：6–18 個月大時手術修補（urethroplasty）

### 11.7 膀胱外翻（Bladder Exstrophy）

- 膀胱黏膜暴露於腹壁外 + 恥骨聯合分離 + epispadias
- 治療：早期（48–72 小時內）手術關閉
- 長期風險：**Adenocarcinoma of the bladder**

## 十二、男性生殖系統急症

### 12.1 睪丸扭轉（Testicular Torsion）

- **泌尿科真正的急症！**
- 好發：新生兒及 **12–18 歲青少年**
- 機制：bell-clapper deformity → tunica vaginalis 附著異常 → 睪丸可在鞘膜內自由旋轉
- 症狀：突發劇烈單側陰囊疼痛、腫脹、噁心嘔吐
- 理學檢查：
    - 患側睪丸上抬（horizontal lie）
    - **Cremasteric reflex 消失**（最有用的理學檢查）
    - **Prehn sign 陰性**（抬高陰囊不能緩解疼痛，與 epididymitis 相反）
- 診斷：**臨床診斷優先**；若有疑慮可做 color Doppler ultrasound → 患側血流減少/消失
    - **不要為了做影像而延誤手術！**
- **黃金時間**：**6 小時內**手術 → 睪丸保留率 ~100%；>12 小時 → 明顯降低；>24 小時 → 幾乎不可挽救
- 治療：**緊急 scrotal exploration + detorsion + orchiopexy（雙側固定！）**
    - 若睪丸已壞死 → orchiectomy + 對側 orchiopexy
    - 對側也需固定（因 bell-clapper deformity 常為雙側）

### 12.2 附睪炎（Epididymitis）

- **最常見的急性陰囊疼痛原因**（整體而言）
- 病原菌：
    - < 35 歲：**Chlamydia trachomatis、Neisseria gonorrhoeae**（STI）
    - \> 35 歲：**E. coli**（UTI 相關菌）
- 症狀：逐漸發作的陰囊疼痛（vs 扭轉為突發）、附睪腫脹壓痛、可有發燒
- **Prehn sign 陽性**（抬高陰囊可緩解疼痛）
- **Cremasteric reflex 存在**
- Doppler 超音波：附睪血流**增加**（vs 扭轉血流減少）
- 治療：
    - STI 相關：Ceftriaxone 250 mg IM + Doxycycline 100 mg BID × 10 天
    - UTI 相關：Fluoroquinolone 或 TMP-SMX

### 12.3 Fournier's Gangrene

- **陰囊/會陰的壞死性筋膜炎**
- 好發：DM、免疫低下、肛周感染、泌尿道感染
- 多為 polymicrobial（aerobic + anaerobic）
- 症狀：劇烈疼痛、陰囊腫脹/壞死、crepitus、全身性 sepsis
- 治療：**緊急手術清創（最重要！）** + 廣效抗生素 + 必要時多次清創
- **睪丸通常可保留**（睪丸血供來自腹部而非陰囊皮膚）

### 12.4 嵌頓性包莖（Paraphimosis）

- 包皮退到冠狀溝後無法回覆 → 靜脈回流受阻 → 水腫 → 動脈灌流下降 → **龜頭壞死**
- **泌尿科急症！**
- 處理：手動復位（squeeze technique + push）→ 失敗 → dorsal slit（背側切開）
- 預防：每次翻包皮後要記得翻回（尤其導尿後）

### 12.5 陰莖異常勃起（Priapism）

| 類型 | 機制 | 特點 | 治療 |
|------|------|------|------|
| **Ischemic（low-flow）** | 靜脈回流受阻 → 血液滯留 | **痛**、黑暗色血、常見於鐮刀細胞貧血、PDE5i 過量；**急症** | Aspiration of blood + **Phenylephrine injection**（α-agonist）→ 失敗 → surgical shunt |
| **Non-ischemic（high-flow）** | 動脈瘻管（通常外傷後） | 不痛、部分勃起 | 觀察 → 必要時 angioembolization |

- **Ischemic priapism > 4 小時需緊急處理** → 延遲會導致 corpus cavernosum 纖維化 → 永久性 ED

## 十三、男性生殖功能與性功能

### 13.1 男性生殖內分泌軸

- **HPG axis（下視丘-腦下垂體-性腺軸）**：
    - 下視丘 → GnRH（脈衝式分泌）→ 腦下垂體前葉
    - **LH → Leydig cells** → 分泌 **Testosterone**
    - **FSH → Sertoli cells** → 支持精子生成（spermatogenesis）；Sertoli cells 同時分泌 **Inhibin B** → 負回饋抑制 FSH
    - Testosterone → 負回饋抑制 GnRH 和 LH
- **外源性睪固酮（exogenous testosterone）會抑制 HPG axis** → GnRH/LH/FSH 下降 → **Sertoli/Leydig cells 活性降低 → 精子生成受損、睪丸萎縮** → 男性不孕！因此有生育需求者**禁用外源性睪固酮**作為治療

### 13.2 勃起功能障礙（Erectile Dysfunction, ED）

- 原因：血管性（最常見—動脈硬化）、神經性（DM neuropathy、手術損傷 cavernous nerve）、內分泌性（低睪固酮）、心因性、藥物性（β-blocker、SSRI、thiazide）
- **PDE5 inhibitor 機轉**：性刺激 → NO 釋放 → 活化 guanylyl cyclase → cGMP↑ → 海綿體平滑肌放鬆 → 血液灌注 → 勃起。PDE5 分解 cGMP → **PDE5 inhibitor（Sildenafil, Tadalafil）抑制 cGMP 降解** → 維持/增強勃起
- **禁與硝酸鹽類（nitrates）併用** → 兩者皆增加 cGMP/NO pathway → 致命性低血壓
- 第二線：Intracavernosal injection（PGE1 / Alprostadil）、vacuum erection device
- 第三線：Penile prosthesis（inflatable 或 malleable）

### 13.3 男性不孕症

- 精液分析基本正常值（WHO 2021）：
    - 精液量 ≥ 1.4 mL
    - 精蟲濃度 ≥ 16 × 10⁶/mL
    - 總活動力 ≥ 42%
    - 正常形態 ≥ 4%
- **精索靜脈曲張（Varicocele）**：可矯治的男性不孕最常見原因
    - 左側佔 90%（左精索靜脈 → 左腎靜脈，直角注入）
    - 站立時觸診如 "bag of worms"、Valsalva 加劇
    - Grade III 或影響生育力 → Varicocelectomy（microsurgical inguinal approach 最佳）
- 無精症（Azoospermia）鑑別：
    - **Obstructive**：FSH 正常、睪丸大小正常 → 常見原因：prior vasectomy、先天性輸精管缺失（CBAVD，與 **CFTR 基因/CF** 相關）
    - **Non-obstructive**：FSH 升高、睪丸萎縮 → 精子生成障礙（Klinefelter's、cryptorchidism、化療後）

### 13.4 射精障礙

#### 早洩（Premature Ejaculation, PE）

- 最常見的男性性功能障礙
- 定義：持續或反覆在陰道插入前、期間或插入後極短時間內射精，伴隨無法控制與心理困擾
- 分類：終生型（primary）vs 後天型（acquired）
- 治療：
    - 行為治療：squeeze technique、stop-start method
    - 藥物：**SSRI**（Dapoxetine 為唯一核准的短效 SSRI；off-label 使用 Paroxetine、Sertraline）→ 延遲射精
    - 外用麻醉劑：Lidocaine/Prilocaine cream → 降低龜頭敏感度
    - PDE5 inhibitor 可作為輔助用藥

#### 逆行性射精（Retrograde Ejaculation）

- 精液逆流入膀胱而非由尿道射出 → 乾性高潮（dry ejaculate）
- 原因：膀胱頸關閉不全 → **TURP 後最常見原因**（50–75%）、DM 神經病變、α-blocker 使用、後腹腔淋巴結廓清術（RPLND）
- 診斷：射精後尿液中發現**精蟲**
- 治療：若有生育需求 → 停用相關藥物；α-agonist（Pseudoephedrine）促進膀胱頸關閉；射精後尿液中回收精蟲行人工生殖

### 13.5 包皮相關

- **Phimosis（包莖）**：包皮無法退至冠狀溝
    - 生理性：大部分男嬰正常，3–5 歲前多會自然消退
    - 病理性：反覆感染（balanoposthitis）、排尿困難、BXO（balanitis xerotica obliterans）
    - 治療：外用類固醇 → 失敗 → circumcision
- **Paraphimosis**：見前述急症

### 13.6 陰莖癌（Penile Cancer）

- 罕見，好發 > 60 歲未割包皮男性
- 風險因子：**包莖/未行包皮環切**（最重要）、HPV 感染（HPV 16, 18）、吸菸、慢性陰莖炎（balanoposthitis）、lichen sclerosus（BXO）
- 病理：> 95% 為 **Squamous Cell Carcinoma（SCC）**
- 癌前病變：Erythroplasia of Queyrat（龜頭/包皮內板的 CIS）、Bowen's disease（陰莖幹的 CIS）、Bowenoid papulosis（HPV 相關，較年輕）、Giant condyloma（Buschke-Löwenstein tumor，verrucous carcinoma）
- 臨床：龜頭或包皮上無痛性腫塊/潰瘍、不癒合的傷口
- 淋巴引流：**腹股溝淋巴結（inguinal LN）** → 骨盆腔淋巴結
- 治療：
    - 原發灶：小/表淺 → 局部切除（glansectomy / partial penectomy）；大/侵犯深部 → partial 或 total penectomy
    - 可觸及的腹股溝淋巴結 → 切片確認後 → **inguinal lymph node dissection**
- 預防：新生兒 circumcision 可顯著降低風險；HPV 疫苗
